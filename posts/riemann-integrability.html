리만 적분가능성 판정 정리

2025년 11월 13일
← 목록으로 돌아가기

1. 서론

어떤 함수가 리만 적분 가능(Riemann Integrable)한지를 판정하는 것은 해석학(Analysis)의 핵심 주제입니다. 모든 함수가 적분 가능한 것은 아니며, 적분가능성을 판정하는 명확한 기준이 필요합니다.

이 글에서는 리만의 적분가능 조건(Riemann's Criterion)과 리베스그의 정리(Lebesgue's Theorem)를 통해 함수의 적분가능성을 판정하는 방법을 학습합니다.

2. 기본 개념

2.1 분할 (Partition)

정의: 분할

닫힌 구간 $[a,b]$의 분할(partition) $P$는 다음을 만족하는 유한 개의 점들의 집합입니다:

$$P = \{x_0, x_1, x_2, \ldots, x_n\}$$

여기서 $a = x_0 < x_1 < x_2 < \cdots < x_n = b$

세분 분할(Refinement): 분할 $P'$이 $P$를 포함하면 ($P \subseteq P'$), $P'$을 $P$의 세분(refinement)이라고 합니다.

2.2 상합과 하합 (Upper and Lower Sums)

정의: 상합과 하합

함수 $f$가 구간 $[a,b]$에서 유계(bounded)라고 하자. 분할 $P = \{x_0, x_1, \ldots, x_n\}$에 대해:

각 소구간에서의 상한과 하한:

$$M_i = \sup_{x \in [x_{i-1}, x_i]} f(x), \quad m_i = \inf_{x \in [x_{i-1}, x_i]} f(x)$$

상합(Upper Sum):

$$U(P, f) = \sum_{i=1}^{n} M_i (x_i - x_{i-1})$$

하합(Lower Sum):

$$L(P, f) = \sum_{i=1}^{n} m_i (x_i - x_{i-1})$$

항상 $L(P, f) \leq U(P, f)$이며, 세분할수록 하합은 증가하고 상합은 감소합니다.

2.3 상적분과 하적분 (Upper and Lower Integrals)

정의: 상적분과 하적분

상적분(Upper Integral):

$$\overline{\int_a^b} f(x) dx = \inf_P U(P, f)$$

모든 가능한 분할에 대한 상합의 하한(infimum)

하적분(Lower Integral):

$$\underline{\int_a^b} f(x) dx = \sup_P L(P, f)$$

모든 가능한 분할에 대한 하합의 상한(supremum)

항상 $\underline{\int_a^b} f(x) dx \leq \overline{\int_a^b} f(x) dx$가 성립합니다.

3. 리만 적분가능성

정의: 리만 적분가능

유계함수 $f : [a,b] \to \mathbb{R}$가 리만 적분가능(Riemann integrable)하다는 것은 상적분과 하적분이 같다는 의미입니다:

$$\underline{\int_a^b} f(x) dx = \overline{\int_a^b} f(x) dx$$

이 경우, 이 공통된 값을 리만 적분(Riemann integral)이라 하고 $\int_a^b f(x) dx$로 표기합니다.

4. 리만의 적분가능 조건

정리 (리만의 적분가능 조건, Riemann's Criterion)

유계함수 $f : [a,b] \to \mathbb{R}$가 리만 적분가능하기 위한 필요충분조건은:

$$\text{임의의 } \varepsilon > 0 \text{에 대해 분할 } P \text{가 존재하여 } U(P, f) - L(P, f) < \varepsilon$$

즉, 상합과 하합의 차이를 임의로 작게 만들 수 있어야 합니다.

증명 개요

필요조건 (→):

$f$가 적분가능하면 $\overline{\int} f = \underline{\int} f$입니다. 상적분과 하적분의 정의에 의해, 임의의 $\varepsilon > 0$에 대해 $U(P, f) - L(P, f) < \varepsilon$인 분할 $P$가 존재합니다.

충분조건 (←):

임의의 $\varepsilon > 0$에 대해 $U(P, f) - L(P, f) < \varepsilon$이면, $L(P, f) \leq \underline{\int} f \leq \overline{\int} f \leq U(P, f)$이므로 $\overline{\int} f - \underline{\int} f < \varepsilon$입니다. $\varepsilon$이 임의이므로 $\overline{\int} f = \underline{\int} f$입니다. ∎

5. 적분가능 함수의 종류

5.1 연속함수

정리

$f$가 닫힌 구간 $[a,b]$에서 연속(continuous)이면, $f$는 리만 적분가능합니다.

이유: 연속함수는 닫힌 구간에서 균일연속(uniformly continuous)이므로, 충분히 세밀한 분할을 취하면 각 소구간에서 $M_i - m_i$를 임의로 작게 만들 수 있습니다.

5.2 단조함수

정리

$f$가 닫힌 구간 $[a,b]$에서 단조(monotone)이면, $f$는 리만 적분가능합니다.

예제 1: 단조증가 함수

$f(x) = \lfloor x \rfloor$ (바닥함수, floor function)를 $[0, 3]$에서 생각합니다. 이 함수는 단조증가이므로 적분가능합니다.

등분할 $P_n$을 취하면:

$$U(P_n, f) - L(P_n, f) = \frac{f(3) - f(0)}{n} \cdot (3-0) = \frac{3 \cdot 3}{n} = \frac{9}{n} \to 0 \text{ as } n \to \infty$$

6. 리베스그의 정리

적분가능성을 불연속점(discontinuities)의 관점에서 특징짓는 강력한 정리가 있습니다.

정의: 측도 0 (Measure Zero)

집합 $E \subset \mathbb{R}$이 측도 0 (measure zero)을 갖는다는 것은:

임의의 $\varepsilon > 0$에 대해, $E$를 덮는 가산개의 열린구간들 $(a_1, b_1), (a_2, b_2), \ldots$가 존재하여

$$\sum_{i=1}^{\infty} (b_i - a_i) < \varepsilon$$

측도 0인 집합의 예

  • 유한 집합: $\{x_1, x_2, \ldots, x_n\}$
  • 가산 집합: 유리수 집합 $\mathbb{Q}$
  • 칸토르 집합 (Cantor set)

측도 0이 아닌 예: 임의의 구간 $(a, b)$는 측도가 $b-a > 0$

정리 (리베스그의 정리, Lebesgue's Theorem)

유계함수 $f : [a,b] \to \mathbb{R}$가 리만 적분가능하기 위한 필요충분조건은:

$f$의 불연속점의 집합이 측도 0이다

즉, $f$가 "거의 모든 곳(almost everywhere)"에서 연속이어야 합니다.

이 정리는 적분가능성을 판정하는 매우 실용적인 기준을 제공합니다.

7. 예제와 응용

예제 2: 적분가능한 함수

다음 함수를 $[0, 1]$에서 정의합니다:

$$f(x) = \begin{cases} 0 & \text{if } x \in \mathbb{Q} \\ 1 & \text{if } x \notin \mathbb{Q} \end{cases}$$

이 함수는 적분 불가능합니다.

이유:

임의의 분할 $P$에 대해, 각 소구간은 유리수와 무리수를 모두 포함하므로:

$$M_i = 1, \quad m_i = 0 \text{ for all } i$$
$$U(P, f) = 1, \quad L(P, f) = 0$$

따라서 $U(P, f) - L(P, f) = 1 > 0$이므로 리만의 조건을 만족하지 않습니다.

또한, $f$는 모든 점에서 불연속이므로 불연속점의 집합이 $[0,1]$이며 측도가 1 > 0입니다.

예제 3: 적분가능한 함수

다음 함수를 $[0, 2]$에서 정의합니다:

$$g(x) = \begin{cases} x & \text{if } x \neq 1 \\ 5 & \text{if } x = 1 \end{cases}$$

이 함수는 적분가능합니다.

이유:

$g$는 $x = 1$에서만 불연속이고, 한 점의 집합 $\{1\}$은 측도 0입니다. 리베스그의 정리에 의해 $g$는 적분가능하며:

$$\int_0^2 g(x) dx = \int_0^2 x dx = \left[\frac{x^2}{2}\right]_0^2 = 2$$

(한 점에서의 함숫값 변경은 적분값에 영향을 주지 않습니다)

예제 4: 가산개의 점에서 불연속

다음 함수를 $[0, 1]$에서 정의합니다:

$$h(x) = \begin{cases} \frac{1}{n} & \text{if } x = \frac{1}{n}, \; n \in \mathbb{N} \\ 0 & \text{otherwise} \end{cases}$$

이 함수는 적분가능합니다.

이유:

불연속점은 $\left\{\frac{1}{n} : n \in \mathbb{N}\right\} \cup \{0\}$로 가산집합이므로 측도 0입니다. 따라서 적분가능하며:

$$\int_0^1 h(x) dx = 0$$

8. 연습문제

문제 1

다음 함수가 $[0, 1]$에서 리만 적분가능한지 판정하시오:

$$f(x) = \begin{cases} x^2 & \text{if } 0 \leq x < 0.5 \\ 1 & \text{if } x = 0.5 \\ x & \text{if } 0.5 < x \leq 1 \end{cases}$$

풀이 1

Step 1: 불연속점 찾기

$x < 0.5$일 때 $\lim_{x \to 0.5^-} f(x) = (0.5)^2 = 0.25$

$x > 0.5$일 때 $\lim_{x \to 0.5^+} f(x) = 0.5$

$f(0.5) = 1$

따라서 $x = 0.5$에서만 불연속입니다.

Step 2: 적분가능성 판정

불연속점은 $\{0.5\}$로 한 점이므로 측도 0입니다. 리베스그의 정리에 의해 $f$는 적분가능합니다.

Step 3: 적분값 계산

$$\begin{align} \int_0^1 f(x) dx &= \int_0^{0.5} x^2 dx + \int_{0.5}^1 x dx \\ &= \left[\frac{x^3}{3}\right]_0^{0.5} + \left[\frac{x^2}{2}\right]_{0.5}^1 \\ &= \frac{(0.5)^3}{3} + \frac{1}{2} - \frac{(0.5)^2}{2} \\ &= \frac{1}{24} + \frac{1}{2} - \frac{1}{8} = \frac{10}{24} = \frac{5}{12} \end{align}$$

문제 2

다음 함수가 $[0, 1]$에서 리만 적분가능한지 판정하시오:

$$g(x) = \begin{cases} 1 & \text{if } x = \frac{1}{2^n}, \; n \in \mathbb{N} \\ 0 & \text{otherwise} \end{cases}$$

풀이 2

Step 1: 불연속점 찾기

$g$는 다음 점들에서만 불연속입니다:

$$D = \left\{\frac{1}{2}, \frac{1}{4}, \frac{1}{8}, \frac{1}{16}, \ldots\right\} \cup \{0\}$$

(수열 $\frac{1}{2^n}$은 0으로 수렴하므로 $x=0$에서도 불연속)

Step 2: 측도 판정

$D$는 가산집합이므로 측도 0입니다.

Step 3: 결론

리베스그의 정리에 의해 $g$는 적분가능하며:

$$\int_0^1 g(x) dx = 0$$

(가산개의 점에서의 함숫값은 적분값에 영향을 주지 않음)

문제 3

분할 $P = \{0, 0.5, 1\}$에 대해 함수 $f(x) = x^2$의 상합과 하합을 계산하시오.

풀이 3

소구간: $[0, 0.5]$와 $[0.5, 1]$

구간 $[0, 0.5]$:

$f(x) = x^2$는 증가함수이므로:

$$m_1 = f(0) = 0, \quad M_1 = f(0.5) = 0.25$$

구간 $[0.5, 1]$:

$$m_2 = f(0.5) = 0.25, \quad M_2 = f(1) = 1$$

하합:

$$L(P, f) = m_1 \cdot 0.5 + m_2 \cdot 0.5 = 0 \cdot 0.5 + 0.25 \cdot 0.5 = 0.125$$

상합:

$$U(P, f) = M_1 \cdot 0.5 + M_2 \cdot 0.5 = 0.25 \cdot 0.5 + 1 \cdot 0.5 = 0.625$$

차이:

$$U(P, f) - L(P, f) = 0.625 - 0.125 = 0.5$$

참고: 실제 적분값은 $\int_0^1 x^2 dx = \frac{1}{3} \approx 0.333$이며, $L(P, f) < \frac{1}{3} < U(P, f)$입니다.

문제 4 (도전)

디리클레 함수(Dirichlet function)를 $[0, 1]$에서 정의합니다:

$$D(x) = \begin{cases} 1 & \text{if } x \in \mathbb{Q} \\ 0 & \text{if } x \notin \mathbb{Q} \end{cases}$$

이 함수가 적분 불가능함을 두 가지 방법으로 증명하시오: (a) 리만의 조건 사용, (b) 리베스그의 정리 사용

풀이 4

(a) 리만의 조건 사용:

임의의 분할 $P = \{x_0, x_1, \ldots, x_n\}$를 생각합니다.

각 소구간 $[x_{i-1}, x_i]$는:

  • 유리수를 포함 (유리수는 조밀)
  • 무리수를 포함 (무리수는 조밀)

따라서:

$$M_i = \sup_{[x_{i-1}, x_i]} D(x) = 1, \quad m_i = \inf_{[x_{i-1}, x_i]} D(x) = 0$$

상합과 하합:

$$U(P, D) = \sum_{i=1}^n 1 \cdot (x_i - x_{i-1}) = 1$$ $$L(P, D) = \sum_{i=1}^n 0 \cdot (x_i - x_{i-1}) = 0$$

따라서:

$$U(P, D) - L(P, D) = 1$$

분할을 아무리 세밀하게 해도 차이가 1이므로, 리만의 조건을 만족하지 않습니다. 적분 불가능.

(b) 리베스그의 정리 사용:

$D$는 모든 점에서 불연속입니다:

  • 유리수 $q$에서: $D(q) = 1$이지만 임의로 가까운 무리수에서 $D = 0$
  • 무리수 $r$에서: $D(r) = 0$이지만 임의로 가까운 유리수에서 $D = 1$

따라서 불연속점의 집합은 $[0, 1]$ 전체이며, 측도는 1 > 0입니다.

리베스그의 정리에 의해 적분 불가능입니다. ∎

9. 중요 성질 정리

리만 적분가능 함수의 성질

$f, g$가 $[a,b]$에서 리만 적분가능하면:

  1. 선형성: $cf + dg$도 적분가능 ($c, d$는 상수)
  2. 곱: $fg$도 적분가능
  3. 절댓값: $|f|$도 적분가능
  4. 최대/최소: $\max(f,g)$, $\min(f,g)$도 적분가능
  5. 합성 (조건부): $h \circ f$가 적분가능 (단, $h$가 연속)

10. 요약

판정 방법 내용 장점
리만의 조건 $U(P, f) - L(P, f) < \varepsilon$ 정의에 가까움, 직접 계산 가능
리베스그의 정리 불연속점의 측도 = 0 실용적, 기하학적 직관
연속성 연속함수는 적분가능 가장 쉬운 판정
단조성 단조함수는 적분가능 불연속점 확인 불필요
핵심 정리:
  • 유계함수가 적분가능 ⟺ 상적분 = 하적분
  • 리만의 조건: 상합과 하합의 차이를 임의로 작게 만들 수 있음
  • 리베스그의 정리: 불연속점의 집합이 측도 0
  • 연속함수, 단조함수는 항상 적분가능
  • 유한개 또는 가산개의 불연속점만 있으면 적분가능
  • 모든 점에서 불연속이면 적분 불가능 (예: 디리클레 함수)
