1. 삼각함수의 미분 공식

삼각함수의 미분은 미적분학에서 가장 기본적이고 중요한 공식입니다. 이 공식들은 극한의 정의를 이용하여 유도할 수 있으며, 다양한 물리학 및 공학 문제에 응용됩니다.

기본 삼각함수의 미분 공식

사인 함수의 미분:

$$\frac{d}{dx}\sin x = \cos x$$

코사인 함수의 미분:

$$\frac{d}{dx}\cos x = -\sin x$$

탄젠트 함수의 미분:

$$\frac{d}{dx}\tan x = \sec^2 x = \frac{1}{\cos^2 x}$$

코탄젠트 함수의 미분:

$$\frac{d}{dx}\cot x = -\csc^2 x = -\frac{1}{\sin^2 x}$$

시컨트 함수의 미분:

$$\frac{d}{dx}\sec x = \sec x \tan x$$

코시컨트 함수의 미분:

$$\frac{d}{dx}\csc x = -\csc x \cot x$$

사인 함수의 미분 유도

미분의 정의를 사용하여 $\sin x$의 도함수를 구합니다:

$$\frac{d}{dx}\sin x = \lim_{h \to 0} \frac{\sin(x+h) - \sin x}{h}$$

덧셈정리 $\sin(x+h) = \sin x \cos h + \cos x \sin h$를 사용하면:

$$= \lim_{h \to 0} \frac{\sin x \cos h + \cos x \sin h - \sin x}{h}$$ $$= \lim_{h \to 0} \frac{\sin x (\cos h - 1) + \cos x \sin h}{h}$$ $$= \lim_{h \to 0} \left[\sin x \cdot \frac{\cos h - 1}{h} + \cos x \cdot \frac{\sin h}{h}\right]$$

다음의 극한값을 사용합니다:

  • $\lim_{h \to 0} \frac{\sin h}{h} = 1$
  • $\lim_{h \to 0} \frac{\cos h - 1}{h} = 0$

따라서:

$$\frac{d}{dx}\sin x = \sin x \cdot 0 + \cos x \cdot 1 = \cos x$$

탄젠트 함수의 미분 유도

$\tan x = \frac{\sin x}{\cos x}$이므로 몫의 미분법을 사용합니다:

$$\frac{d}{dx}\tan x = \frac{d}{dx}\left(\frac{\sin x}{\cos x}\right)$$ $$= \frac{\cos x \cdot \cos x - \sin x \cdot (-\sin x)}{\cos^2 x}$$ $$= \frac{\cos^2 x + \sin^2 x}{\cos^2 x}$$ $$= \frac{1}{\cos^2 x} = \sec^2 x$$

2. 삼각함수의 적분 공식

삼각함수의 적분은 미분의 역연산입니다. 미분 공식을 알고 있다면 적분 공식은 자연스럽게 유도할 수 있습니다.

기본 삼각함수의 적분 공식

사인 함수의 적분:

$$\int \sin x \, dx = -\cos x + C$$

코사인 함수의 적분:

$$\int \cos x \, dx = \sin x + C$$

탄젠트 함수의 적분:

$$\int \tan x \, dx = -\ln|\cos x| + C = \ln|\sec x| + C$$

코탄젠트 함수의 적분:

$$\int \cot x \, dx = \ln|\sin x| + C$$

시컨트 제곱 함수의 적분:

$$\int \sec^2 x \, dx = \tan x + C$$

코시컨트 제곱 함수의 적분:

$$\int \csc^2 x \, dx = -\cot x + C$$

시컨트와 탄젠트 곱의 적분:

$$\int \sec x \tan x \, dx = \sec x + C$$

코시컨트와 코탄젠트 곱의 적분:

$$\int \csc x \cot x \, dx = -\csc x + C$$

탄젠트 함수의 적분 유도

$\tan x = \frac{\sin x}{\cos x}$를 이용합니다:

$$\int \tan x \, dx = \int \frac{\sin x}{\cos x} \, dx$$

$u = \cos x$로 치환하면 $du = -\sin x \, dx$이므로:

$$= -\int \frac{1}{u} \, du = -\ln|u| + C$$ $$= -\ln|\cos x| + C = \ln|\sec x| + C$$

3. 공식 요약표

함수 미분 적분
$\sin x$ $\cos x$ $-\cos x + C$
$\cos x$ $-\sin x$ $\sin x + C$
$\tan x$ $\sec^2 x$ $-\ln|\cos x| + C$
$\cot x$ $-\csc^2 x$ $\ln|\sin x| + C$
$\sec x$ $\sec x \tan x$ $\ln|\sec x + \tan x| + C$
$\csc x$ $-\csc x \cot x$ $-\ln|\csc x + \cot x| + C$

4. 연습문제

연습문제 1

다음 함수를 미분하시오:

(1) $f(x) = x^2 \sin x$

(2) $g(x) = \frac{\cos x}{x}$

풀이

(1) $f(x) = x^2 \sin x$의 미분

곱의 미분법을 사용합니다:

$$f'(x) = (x^2)' \sin x + x^2 (\sin x)'$$ $$= 2x \sin x + x^2 \cos x$$

(2) $g(x) = \frac{\cos x}{x}$의 미분

몫의 미분법을 사용합니다:

$$g'(x) = \frac{(\cos x)' \cdot x - \cos x \cdot (x)'}{x^2}$$ $$= \frac{-\sin x \cdot x - \cos x \cdot 1}{x^2}$$ $$= \frac{-x\sin x - \cos x}{x^2}$$

답:
(1) $f'(x) = 2x\sin x + x^2\cos x$
(2) $g'(x) = \frac{-x\sin x - \cos x}{x^2}$

연습문제 2

다음 정적분을 계산하시오:

(1) $\displaystyle\int_0^{\pi/2} \sin x \, dx$

(2) $\displaystyle\int_0^{\pi/4} \sec^2 x \, dx$

풀이

(1) $\displaystyle\int_0^{\pi/2} \sin x \, dx$

부정적분: $\int \sin x \, dx = -\cos x + C$

$$\int_0^{\pi/2} \sin x \, dx = \left[-\cos x\right]_0^{\pi/2}$$ $$= -\cos\frac{\pi}{2} - (-\cos 0)$$ $$= -0 + 1 = 1$$

(2) $\displaystyle\int_0^{\pi/4} \sec^2 x \, dx$

부정적분: $\int \sec^2 x \, dx = \tan x + C$

$$\int_0^{\pi/4} \sec^2 x \, dx = \left[\tan x\right]_0^{\pi/4}$$ $$= \tan\frac{\pi}{4} - \tan 0$$ $$= 1 - 0 = 1$$

답:
(1) $1$
(2) $1$

5. 활용

삼각함수의 미분과 적분은 다음과 같은 분야에서 널리 사용됩니다:

  • 물리학: 단진동, 파동, 전자기학에서 주기적 운동 분석
  • 공학: 신호처리, 회로 해석, 구조역학
  • 천문학: 행성의 궤도 계산, 천체의 주기적 운동
  • 음향학: 소리의 파동 분석, 푸리에 변환
  • 확률론: 정규분포, 주기적 확률 과정

이러한 공식들을 잘 이해하고 활용하면 다양한 실생활 문제와 고급 수학 문제를 효과적으로 해결할 수 있습니다.
