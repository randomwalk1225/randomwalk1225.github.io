수학 I — 지수·로그함수

고등학교 2학년  |  2022 개정 교육과정  |  지수법칙 · 로그법칙 · 지수함수 · 로그함수

지수함수와 로그함수는 자연과학, 경제학, 공학 전반에서 쓰이는 핵심 함수입니다. 수능에서는 두 함수의 그래프 해석, 지수·로그 방정식·부등식이 매년 출제됩니다. 개념을 정확히 이해하고 계산 실수를 줄이는 것이 고득점의 핵심입니다.

1. 지수의 확장과 지수법칙

1-1. 거듭제곱과 거듭제곱근

정의: 거듭제곱근

$n$이 2 이상의 자연수, $a$가 실수일 때, $x^n = a$를 만족하는 $x$를 $a$의 $n$제곱근이라 한다.

  • $n$이 홀수: 실수 $n$제곱근은 하나 → $\sqrt[n]{a}$
  • $n$이 짝수, $a > 0$: 양의 실수 $n$제곱근 $\sqrt[n]{a}$, 음의 실수 $n$제곱근 $-\sqrt[n]{a}$
  • $n$이 짝수, $a = 0$: $x = 0$
  • $n$이 짝수, $a < 0$: 실수 $n$제곱근 없음

지수법칙 (지수가 유리수·실수로 확장)

$a > 0, b > 0$이고 $m, n$이 유리수(또는 실수)일 때:

  • $a^m \cdot a^n = a^{m+n}$
  • $a^m \div a^n = a^{m-n}$
  • $(a^m)^n = a^{mn}$
  • $(ab)^n = a^n b^n$
  • $a^0 = 1$, $a^{-n} = \dfrac{1}{a^n}$, $a^{1/n} = \sqrt[n]{a}$

예제 1. 지수법칙 계산

다음을 계산하여라: $\dfrac{8^{2/3} \times 27^{1/3}}{6^{1/3}}$

풀이

$8^{2/3} = (2^3)^{2/3} = 2^2 = 4$

$27^{1/3} = (3^3)^{1/3} = 3$

$6^{1/3} = (2 \cdot 3)^{1/3} = 2^{1/3} \cdot 3^{1/3}$

$\therefore \dfrac{4 \times 3}{2^{1/3} \cdot 3^{1/3}} = \dfrac{12}{6^{1/3}} = 12 \cdot 6^{-1/3} = 2 \cdot 6^{2/3}$

예제 2. 유리지수 변환

$a^{1/2} + a^{-1/2} = 3$일 때, $a + a^{-1}$의 값을 구하여라.

풀이

$(a^{1/2} + a^{-1/2})^2 = a + 2 + a^{-1} = 9$

$\therefore a + a^{-1} = 7$

2. 로그(Logarithm)

정의: 로그

$a > 0,\; a \neq 1,\; N > 0$일 때:

$$a^x = N \quad \Leftrightarrow \quad x = \log_a N$$

$a$를 밑(base), $N$을 진수(argument)라 한다.

로그의 성질 (로그법칙)

$a > 0, a \neq 1, M > 0, N > 0$일 때:

  • $\log_a 1 = 0$, $\log_a a = 1$
  • $\log_a MN = \log_a M + \log_a N$
  • $\log_a \dfrac{M}{N} = \log_a M - \log_a N$
  • $\log_a M^k = k\log_a M$ (단, $k$는 실수)
  • 밑 변환 공식: $\log_a b = \dfrac{\log_c b}{\log_c a}$ (단, $c > 0, c \neq 1$)

예제 3. 로그 계산

$\log_2 3 = a$, $\log_2 5 = b$로 놓을 때, $\log_2 \sqrt{75}$를 $a, b$로 나타내어라.

풀이

$\log_2 \sqrt{75} = \dfrac{1}{2}\log_2 75 = \dfrac{1}{2}\log_2(3 \times 5^2)$

$= \dfrac{1}{2}(\log_2 3 + 2\log_2 5) = \dfrac{1}{2}(a + 2b) = \dfrac{a+2b}{2}$

예제 4. 밑 변환 공식 활용

$\log_2 3 \times \log_3 5 \times \log_5 8$의 값을 구하여라.

풀이

밑 변환 공식으로 모두 상용로그로 변환:

$= \dfrac{\log 3}{\log 2} \times \dfrac{\log 5}{\log 3} \times \dfrac{\log 8}{\log 5} = \dfrac{\log 8}{\log 2} = \dfrac{3\log 2}{\log 2} = 3$

주의: 로그의 정의역 조건

$\log_a f(x)$에서 반드시 확인해야 할 조건:

  • 밑 조건: $a > 0,\; a \neq 1$
  • 진수 조건: $f(x) > 0$

방정식·부등식 풀이 후 반드시 이 조건을 대입해 검증한다.

3. 지수함수

지수함수의 그래프와 성질

$y = a^x$ ($a > 0, a \neq 1$)

성질$a > 1$$0 < a < 1$
증감순증가순감소
정의역실수 전체 $(-\infty, \infty)$
치역양의 실수 $(0, \infty)$
점근선$x$축 ($y = 0$)
기준점$(0, 1)$을 지남

예제 5. 지수방정식

$4^x - 3 \cdot 2^x - 4 = 0$을 풀어라.

풀이

$t = 2^x$ $(t > 0)$으로 치환: $t^2 - 3t - 4 = 0$

$(t-4)(t+1) = 0 \Rightarrow t = 4$ 또는 $t = -1$

$t > 0$이므로 $t = 4$, 즉 $2^x = 4 = 2^2 \Rightarrow x = 2$

예제 6. 지수부등식

$\left(\dfrac{1}{3}\right)^{x-1} > 9^{2-x}$를 풀어라.

풀이

$3^{-(x-1)} > 3^{2(2-x)}$

$-(x-1) > 2(2-x)$

$-x+1 > 4-2x$

$x > 3$

밑 $\dfrac{1}{3} < 1$이므로 부등호 방향이 바뀌는 것에 주의: 밑을 3으로 통일하면 지수의 부등호 방향 그대로.

4. 로그함수

로그함수의 그래프와 성질

$y = \log_a x$ ($a > 0, a \neq 1$) — 지수함수 $y = a^x$의 역함수

성질$a > 1$$0 < a < 1$
증감순증가순감소
정의역양의 실수 $(0, \infty)$
치역실수 전체 $(-\infty, \infty)$
점근선$y$축 ($x = 0$)
기준점$(1, 0)$을 지남

예제 7. 로그방정식

$\log_2(x+1) + \log_2(x+3) = 3$을 풀어라.

풀이

$\log_2\{(x+1)(x+3)\} = 3$

$(x+1)(x+3) = 8$

$x^2 + 4x + 3 = 8 \Rightarrow x^2 + 4x - 5 = 0$

$(x+5)(x-1) = 0 \Rightarrow x = -5$ 또는 $x = 1$

진수 조건: $x+1 > 0, x+3 > 0 \Rightarrow x > -1$

따라서 $x = 1$

예제 8. 로그부등식

$\log_3(x^2 - 3x + 3) < 1$을 풀어라.

풀이

밑 $3 > 1$이므로 부등호 방향 그대로:

$x^2 - 3x + 3 < 3^1 = 3$

$x^2 - 3x < 0 \Rightarrow x(x-3) < 0 \Rightarrow 0 < x < 3$

진수 조건: $x^2 - 3x + 3 > 0$ — 판별식 $= 9 - 12 = -3 < 0$이므로 항상 성립.

따라서 $0 < x < 3$

예제 9. 상용로그의 활용

$\log_{10} 2 = 0.3010$으로 주어질 때, $2^{20}$은 몇 자리 자연수인가?

풀이

$\log_{10} 2^{20} = 20 \times 0.3010 = 6.020$

$6 \leq 6.020 < 7$이므로 $10^6 \leq 2^{20} < 10^7$

따라서 $2^{20}$은 7자리 자연수이다.

수능 핵심 전략

  • 치환: 지수방정식 $a^{2x} - a^x \cdot k - m = 0$은 $t = a^x$으로 치환 후 이차방정식 풀기
  • 공통 밑: 서로 다른 밑을 공통 밑으로 통일하는 것이 핵심
  • 진수 조건: 로그 방정식·부등식의 최후 검증 단계, 절대 생략 금지
  • 역함수 관계: $y = a^x$와 $y = \log_a x$는 $y = x$에 대해 대칭 — 그래프 문제 활용

연습 문제

  1. $27^{1/3} \times 9^{3/2} \div 3^2$를 계산하여라.
  2. $\log_3 5 = a$, $\log_3 7 = b$로 놓을 때, $\log_3 \dfrac{35}{9}$를 $a, b$로 나타내어라.
  3. $9^x - 4 \cdot 3^x + 3 = 0$을 풀어라.
  4. $\log_2(x-1) + \log_2(x+1) = 3$을 풀어라.
  5. $\log_{1/2}(x^2 - 2x) \geq -2$를 풀어라.
