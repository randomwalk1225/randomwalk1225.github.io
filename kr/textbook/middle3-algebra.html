중학교 3학년 — 이차방정식과 이차함수

중학교 3학년  |  2022 개정 교육과정  |  이차방정식 · 이차함수

이 단원에서는 이차방정식의 다양한 풀이법(인수분해, 완전제곱식, 근의 공식)과 이차함수의 그래프 성질을 학습합니다. 이 내용은 고등학교 수학(공통수학 1)에서 이차함수·이차방정식의 심화로 이어지는 핵심 단원입니다.

1. 이차방정식

1-1. 이차방정식의 뜻과 해

정의: 이차방정식

이차방정식: $ax^2 + bx + c = 0$ ($a \neq 0$) 형태의 방정식.

  • $x^2 - 5x + 6 = 0$ — 이차방정식 ✓
  • $x^2 = 4$ — 이차방정식 (정리하면 $x^2 - 4 = 0$) ✓
  • $x^2 - x^2 + 3x = 0$ — 이차방정식 아님 (이차항 소거) ✗

해(근): 이차방정식을 참이 되게 하는 $x$의 값. 이차방정식은 일반적으로 해가 2개.

1-2. 인수분해를 이용한 풀이

핵심 인수분해 공식

$$x^2 + (a+b)x + ab = (x+a)(x+b)$$ $$x^2 - a^2 = (x+a)(x-a) \quad \text{(합차공식)}$$ $$x^2 \pm 2ax + a^2 = (x \pm a)^2 \quad \text{(완전제곱식)}$$

예제 1: 인수분해 풀이 (기본)

$x^2 - 5x + 6 = 0$을 풀어라.

합이 $-5$, 곱이 $6$인 두 수: $-2$와 $-3$

$(x-2)(x-3) = 0$

$x = 2$ 또는 $x = 3$

예제 2: 완전제곱식

$x^2 + 6x + 9 = 0$을 풀어라.

$(x+3)^2 = 0$

$x = -3$ (중근)

예제 3: 합차공식

$4x^2 - 25 = 0$을 풀어라.

$(2x+5)(2x-5) = 0$

$x = -\dfrac{5}{2}$ 또는 $x = \dfrac{5}{2}$

예제 4: 계수가 있는 이차방정식

$2x^2 - 5x - 3 = 0$을 풀어라.

$(2x + 1)(x - 3) = 0$

$x = -\dfrac{1}{2}$ 또는 $x = 3$

1-3. 완전제곱식을 이용한 풀이

완전제곱식으로 변형하는 방법

$x^2 + px = q$ 형태에서 양변에 $\left(\dfrac{p}{2}\right)^2$을 더하면:

$$\left(x + \frac{p}{2}\right)^2 = q + \left(\frac{p}{2}\right)^2$$

예제 5: 완전제곱식 변형

$x^2 + 4x - 3 = 0$을 완전제곱식으로 풀어라.

$x^2 + 4x = 3$

양변에 $\left(\dfrac{4}{2}\right)^2 = 4$를 더하면:

$(x+2)^2 = 7$

$x + 2 = \pm\sqrt{7}$

$x = -2 \pm \sqrt{7}$

1-4. 근의 공식

근의 공식

$ax^2 + bx + c = 0$ ($a \neq 0$)의 해:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

$b^2 - 4ac$를 판별식 $D$라 한다.

  • $D > 0$: 서로 다른 두 실근
  • $D = 0$: 중근 (두 근이 같음)
  • $D < 0$: 실근 없음 (허근 — 고등학교에서 학습)

예제 6: 근의 공식 적용

$2x^2 - 3x - 2 = 0$을 근의 공식으로 풀어라.

$a=2,\; b=-3,\; c=-2$

$x = \dfrac{3 \pm \sqrt{9 + 16}}{4} = \dfrac{3 \pm \sqrt{25}}{4} = \dfrac{3 \pm 5}{4}$

$x = 2$ 또는 $x = -\dfrac{1}{2}$

예제 7: 무리수 근

$x^2 - 6x + 7 = 0$을 풀어라.

$D = 36 - 28 = 8 > 0$ → 서로 다른 두 실근

$x = \dfrac{6 \pm \sqrt{8}}{2} = \dfrac{6 \pm 2\sqrt{2}}{2} = 3 \pm \sqrt{2}$

$x = 3 + \sqrt{2}$ 또는 $x = 3 - \sqrt{2}$

1-5. 이차방정식의 활용

예제 8: 넓이 문제

가로가 세로보다 4 cm 더 긴 직사각형의 넓이가 45 cm²이다. 가로의 길이를 구하여라.

세로를 $x$ cm로 놓으면 가로 $= x + 4$ cm

$x(x+4) = 45$

$x^2 + 4x - 45 = 0$

$(x+9)(x-5) = 0$

$x = 5$ (양수이므로 $x = -9$ 제외)

가로: 9 cm

2. 이차함수

2-1. 이차함수의 뜻과 기본 그래프

정의: 이차함수

$f(x) = ax^2 + bx + c$ ($a \neq 0$) 형태의 함수.

가장 기본적인 이차함수: $y = x^2$

  • 꼭짓점: $(0, 0)$
  • 축의 방정식: $x = 0$ (y축)
  • $a > 0$: 아래로 볼록 (포물선이 위로 열림)
  • $a < 0$: 위로 볼록 (포물선이 아래로 열림)

2-2. $y = a(x-p)^2 + q$ 형태 (표준형)

이차함수의 표준형 성질

$y = a(x-p)^2 + q$에서:

성질값
꼭짓점$(p,\; q)$
축의 방정식$x = p$
볼록 방향$a>0$: 아래로 볼록 / $a<0$: 위로 볼록
최솟값 ($a>0$)$y = q$ (꼭짓점에서)
최댓값 ($a<0$)$y = q$ (꼭짓점에서)

예제 9: 표준형에서 꼭짓점과 축 찾기

$y = 2(x-3)^2 - 5$의 꼭짓점과 축의 방정식을 구하여라.

꼭짓점: $(3, -5)$

축의 방정식: $x = 3$

$a = 2 > 0$이므로 아래로 볼록, 최솟값 $= -5$

예제 10: 일반형 → 표준형 변환

$y = x^2 - 4x + 1$을 표준형으로 변환하고 꼭짓점을 구하여라.

$y = (x^2 - 4x + 4) - 4 + 1$

$y = (x-2)^2 - 3$

꼭짓점: $(2, -3)$, 축: $x = 2$

예제 11: 이차함수의 최솟값 활용

$y = -x^2 + 6x - 5$의 최댓값을 구하여라.

$y = -(x^2 - 6x) - 5$

$= -(x^2 - 6x + 9) + 9 - 5$

$= -(x-3)^2 + 4$

$a = -1 < 0$이므로 위로 볼록, $x = 3$에서 최댓값.

최댓값: $y = 4$

2-3. 이차함수와 이차방정식의 관계

이차함수 그래프와 $x$축의 관계

이차함수 $y = ax^2 + bx + c$의 그래프와 $x$축의 교점은 이차방정식 $ax^2 + bx + c = 0$의 해에 해당한다.

판별식 $D = b^2 - 4ac$그래프와 $x$축의 관계
$D > 0$두 점에서 교차
$D = 0$한 점에서 접함 (꼭짓점이 $x$축 위)
$D < 0$교점 없음 ($x$축과 만나지 않음)

예제 12: 이차함수의 $x$절편 구하기

$y = x^2 - 3x - 10$의 $x$절편을 구하여라.

$y = 0$: $x^2 - 3x - 10 = 0$

$(x-5)(x+2) = 0$

$x$절편: $x = 5$ 또는 $x = -2$

연습 문제

  1. $x^2 - 7x + 12 = 0$을 인수분해로 풀어라.
  2. $3x^2 + x - 2 = 0$을 근의 공식으로 풀어라.
  3. $y = 3(x+1)^2 - 4$의 꼭짓점과 축의 방정식을 구하여라.
  4. $y = x^2 + 2x - 3$을 표준형으로 변환하고 꼭짓점을 구하여라.
  5. 연속하는 두 자연수의 곱이 72일 때, 두 자연수를 구하여라.

시험 대비 핵심 정리

  • 인수분해 가능 여부: 판별식 $D = b^2 - 4ac$가 완전제곱수이면 인수분해 가능
  • 근의 공식: 인수분해가 안 될 때 사용하는 만능 풀이법
  • 꼭짓점: 표준형 $y = a(x-p)^2 + q$에서 $(p, q)$
  • 완전제곱식 변환: $x^2 + bx$에서 $\left(\dfrac{b}{2}\right)^2$을 더하고 빼기
  • 최솟값/최댓값: 꼭짓점의 $y$좌표이고, $a > 0$이면 최솟값, $a < 0$이면 최댓값
