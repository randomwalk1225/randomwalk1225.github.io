수학 II — 미분법

고등학교 2학년  |  2022 개정 교육과정  |  미분계수 · 도함수 · 접선 · 극값 · 최대·최솟값

미분(differentiation)은 함수가 변화하는 순간의 비율(순간변화율)을 구하는 도구입니다. 물리의 속도·가속도, 경제학의 한계비용이 모두 미분 개념입니다. 수능에서는 도함수를 이용한 극값, 최대·최솟값, 접선의 방정식 문제가 빠짐없이 출제됩니다.

1. 미분계수와 도함수

정의: 미분계수

함수 $f(x)$의 $x = a$에서의 미분계수:

$$f'(a) = \lim_{h \to 0}\frac{f(a+h) - f(a)}{h} = \lim_{x \to a}\frac{f(x) - f(a)}{x - a}$$

기하학적 의미: 곡선 $y = f(x)$ 위의 점 $(a, f(a))$에서의 접선의 기울기

정의: 도함수

$f'(x) = \displaystyle\lim_{h \to 0}\dfrac{f(x+h) - f(x)}{h}$

표기: $f'(x) = y' = \dfrac{dy}{dx} = \dfrac{d}{dx}f(x)$

미분법 공식 (다항함수)

  • $(c)' = 0$ (상수의 도함수)
  • $(x^n)' = nx^{n-1}$ ($n$은 실수)
  • $(cf)' = cf'$
  • $(f \pm g)' = f' \pm g'$
  • $(fg)' = f'g + fg'$ (곱의 미분법)
  • $\left(\dfrac{f}{g}\right)' = \dfrac{f'g - fg'}{g^2}$ (몫의 미분법, $g \neq 0$)

예제 1. 도함수 계산

$f(x) = 3x^4 - 2x^3 + 5x - 7$의 도함수를 구하여라.

풀이

$f'(x) = 12x^3 - 6x^2 + 5$

예제 2. 극한으로 미분계수 계산

$\displaystyle\lim_{h \to 0}\dfrac{(2+h)^3 - 8}{h}$를 구하여라.

풀이

이 식은 $f(x) = x^3$의 $x = 2$에서의 미분계수:

$f'(2) = 3x^2\big|_{x=2} = 3 \cdot 4 = 12$

2. 접선의 방정식

접선의 방정식

곡선 $y = f(x)$ 위의 점 $(a, f(a))$에서의 접선의 방정식:

$$y - f(a) = f'(a)(x - a)$$

예제 3. 접선의 방정식

곡선 $y = x^3 - 3x + 2$ 위의 점 $(1, 0)$에서의 접선의 방정식을 구하여라.

풀이

$y' = 3x^2 - 3$, 기울기: $y'(1) = 3 - 3 = 0$

접선: $y - 0 = 0(x - 1) \Rightarrow y = 0$ (수평 접선)

예제 4. 기울기가 주어진 접선

곡선 $y = x^3 - 3x^2 + 1$에서 기울기가 9인 접선의 방정식을 구하여라.

풀이

$y' = 3x^2 - 6x = 9 \Rightarrow 3x^2 - 6x - 9 = 0 \Rightarrow x^2 - 2x - 3 = 0$

$(x-3)(x+1) = 0 \Rightarrow x = 3$ 또는 $x = -1$

$x=3$: $y = 27 - 27 + 1 = 1$, 접선: $y - 1 = 9(x-3) \Rightarrow y = 9x - 26$

$x=-1$: $y = -1 - 3 + 1 = -3$, 접선: $y + 3 = 9(x+1) \Rightarrow y = 9x + 6$

3. 함수의 증가·감소와 극값

증가·감소 판별

  • $f'(x) > 0$인 구간: $f$는 증가
  • $f'(x) < 0$인 구간: $f$는 감소
  • $f'(a) = 0$인 $x = a$: 극값 후보점

극값 판별: $f'(a) = 0$일 때, $x = a$ 근방에서 $f'$의 부호 변화:

  • 양(+) → 음(-): 극대 (local maximum), 극댓값 $f(a)$
  • 음(-) → 양(+): 극소 (local minimum), 극솟값 $f(a)$
  • 부호 변화 없음: 극값 아님

예제 5. 극값 구하기

$f(x) = x^3 - 3x^2 - 9x + 5$의 극댓값과 극솟값을 구하여라.

풀이

$f'(x) = 3x^2 - 6x - 9 = 3(x^2 - 2x - 3) = 3(x-3)(x+1)$

$f'(x) = 0$: $x = -1, 3$

증감표:

$x$$\cdots$$-1$$\cdots$$3$$\cdots$
$f'(x)$$+$$0$$-$$0$$+$
$f(x)$↗극대↘극소↗

극댓값: $f(-1) = -1 - 3 + 9 + 5 = 10$

극솟값: $f(3) = 27 - 27 - 27 + 5 = -22$

4. 최대·최솟값과 활용

닫힌 구간에서의 최대·최솟값

$f$가 $[a, b]$에서 연속이면 최댓값과 최솟값이 반드시 존재하며, 다음 중 하나에서 나타난다:

  • 극값이 되는 점 (내부)
  • 양 끝점 $x = a$ 또는 $x = b$

예제 6. 닫힌 구간에서 최대·최솟값

$f(x) = x^3 - 3x + 2$의 $[-2, 3]$에서 최댓값과 최솟값을 구하여라.

풀이

$f'(x) = 3x^2 - 3 = 3(x-1)(x+1) = 0 \Rightarrow x = \pm 1$

후보점 함수값 비교:

$f(-2) = -8 + 6 + 2 = 0$

$f(-1) = -1 + 3 + 2 = 4$ (극대)

$f(1) = 1 - 3 + 2 = 0$ (극소)

$f(3) = 27 - 9 + 2 = 20$

최댓값: $f(3) = 20$, 최솟값: $f(-2) = f(1) = 0$

예제 7. 속도·가속도 (미분 응용)

시각 $t$에서 점 P의 위치가 $x(t) = t^3 - 6t^2 + 9t$일 때, $t = 2$에서 점 P의 속도와 가속도를 구하여라.

풀이

속도: $v(t) = x'(t) = 3t^2 - 12t + 9$

$v(2) = 12 - 24 + 9 = -3$

가속도: $a(t) = v'(t) = 6t - 12$

$a(2) = 12 - 12 = 0$

예제 8. 함수의 그래프 개형

$f(x) = x^4 - 4x^3$의 변곡점을 구하여라.

풀이

$f'(x) = 4x^3 - 12x^2$

$f''(x) = 12x^2 - 24x = 12x(x-2)$

$f''(x) = 0$: $x = 0, 2$

$f''$ 부호 변화 확인: $x = 0$ 주변 양→음(변곡점), $x = 2$ 주변 음→양(변곡점)

변곡점: $(0, 0)$ 및 $(2, -16)$

수능 핵심 전략

  • 증감표: 극값 문제는 반드시 증감표를 그려 부호 변화를 체계적으로 확인
  • 극한으로 미분계수: $\lim_{h\to 0}\frac{f(a+h)-f(a)}{h}$ 꼴 극한은 미분계수 공식으로 즉시 풀기
  • 닫힌 구간 최대·최소: 극값 + 끝점 모두 계산, 비교
  • 속도·가속도: 위치 $x(t)$ → 미분 → 속도 $v(t)$ → 미분 → 가속도 $a(t)$

연습 문제

  1. $f(x) = 2x^3 - 9x^2 + 12x - 4$의 극댓값과 극솟값을 구하여라.
  2. 곡선 $y = x^2 - 3x + 5$ 위의 점 $(2, 3)$에서의 접선과 법선의 방정식을 구하여라.
  3. $f(x) = x^3 - 6x^2 + 9x + 1$의 $[0, 5]$에서 최댓값과 최솟값을 구하여라.
  4. $\displaystyle\lim_{x \to 1}\dfrac{x^5 - 1}{x - 1}$을 미분계수를 이용해 구하여라.
  5. 위치함수 $x(t) = t^3 - 3t^2 + 2$에서 점 P가 정지하는 시각과 그 위치를 구하여라.
