수학 II — 함수의 극한과 연속

고등학교 2학년  |  2022 개정 교육과정  |  극한값 · 극한의 성질 · 연속함수 · 중간값 정리

극한(limit)은 미분과 적분의 기초가 되는 개념입니다. 함수가 특정 점에 가까워질 때 어떤 값에 수렴하는지를 분석하는 방법을 익히면, 미적분 전체를 논리적으로 이해할 수 있습니다. 수능에서는 극한값 계산, 연속성 판별, 미분계수가 매년 출제됩니다.

1. 함수의 극한

정의: 함수의 극한값

$x$가 $a$에 한없이 가까워질 때 $f(x)$가 일정한 값 $L$에 한없이 가까워지면:

$$\lim_{x \to a} f(x) = L$$

좌극한·우극한:

  • 좌극한: $\displaystyle\lim_{x \to a^-} f(x)$ (왼쪽에서 접근)
  • 우극한: $\displaystyle\lim_{x \to a^+} f(x)$ (오른쪽에서 접근)
  • 극한값 존재 조건: 좌극한 = 우극한 = $L$

극한의 성질

$\displaystyle\lim_{x \to a}f(x) = L$, $\displaystyle\lim_{x \to a}g(x) = M$이면:

  • $\displaystyle\lim_{x \to a}\{f(x) \pm g(x)\} = L \pm M$
  • $\displaystyle\lim_{x \to a}f(x)g(x) = LM$
  • $\displaystyle\lim_{x \to a}\dfrac{f(x)}{g(x)} = \dfrac{L}{M}$ (단, $M \neq 0$)
  • $\displaystyle\lim_{x \to a}cf(x) = cL$ (단, $c$는 상수)

예제 1. 다항함수의 극한

$\displaystyle\lim_{x \to 2}(3x^2 - 5x + 1)$을 구하여라.

풀이

다항함수는 연속이므로 직접 대입:

$= 3(4) - 5(2) + 1 = 12 - 10 + 1 = 3$

예제 2. $\dfrac{0}{0}$ 꼴 극한 — 인수분해

$\displaystyle\lim_{x \to 3}\dfrac{x^2 - 9}{x - 3}$을 구하여라.

풀이

$x \neq 3$에서 인수분해:

$\dfrac{x^2 - 9}{x - 3} = \dfrac{(x+3)(x-3)}{x-3} = x + 3$

$\displaystyle\lim_{x \to 3}(x+3) = 6$

예제 3. $\dfrac{0}{0}$ 꼴 극한 — 유리화

$\displaystyle\lim_{x \to 4}\dfrac{\sqrt{x} - 2}{x - 4}$를 구하여라.

풀이

분자를 유리화:

$\dfrac{\sqrt{x}-2}{x-4} \cdot \dfrac{\sqrt{x}+2}{\sqrt{x}+2} = \dfrac{x-4}{(x-4)(\sqrt{x}+2)} = \dfrac{1}{\sqrt{x}+2}$

$\displaystyle\lim_{x \to 4}\dfrac{1}{\sqrt{x}+2} = \dfrac{1}{4} = \dfrac{1}{4}$

예제 4. 무한대의 극한 — 최고차항으로 나누기

$\displaystyle\lim_{x \to \infty}\dfrac{3x^2 + 2x - 1}{2x^2 - 5}$를 구하여라.

풀이

분모·분자를 $x^2$으로 나누기:

$= \displaystyle\lim_{x \to \infty}\dfrac{3 + \frac{2}{x} - \frac{1}{x^2}}{2 - \frac{5}{x^2}} = \dfrac{3 + 0 - 0}{2 - 0} = \dfrac{3}{2}$

주의: $\dfrac{\infty}{\infty}$ 꼴 극한의 비교

$\displaystyle\lim_{x \to \infty}\dfrac{f(x)}{g(x)}$에서 ($f, g$는 다항식):

  • $\deg(f) = \deg(g)$: 최고차항 계수의 비
  • $\deg(f) < \deg(g)$: 극한값 = 0
  • $\deg(f) > \deg(g)$: 극한값 = $\pm\infty$ (발산)

2. 연속함수

정의: 함수의 연속

함수 $f(x)$가 점 $x = a$에서 연속이려면 다음 세 조건 모두 성립해야 한다:

  1. $f(a)$가 정의되어 있다.
  2. $\displaystyle\lim_{x \to a}f(x)$가 존재한다.
  3. $\displaystyle\lim_{x \to a}f(x) = f(a)$

하나라도 성립하지 않으면 $x = a$에서 불연속이다.

예제 5. 연속 조건으로 상수 결정

$f(x) = \begin{cases} x^2 + ax + 1 & (x < 2) \\ bx - 3 & (x \geq 2) \end{cases}$이 $x = 2$에서 연속이고 $\displaystyle\lim_{x \to 2}f(x) = 5$일 때, $a + b$를 구하여라.

풀이

우극한: $f(2) = 2b - 3 = 5 \Rightarrow b = 4$

좌극한: $\displaystyle\lim_{x \to 2^-}(x^2 + ax + 1) = 4 + 2a + 1 = 5 + 2a = 5 \Rightarrow a = 0$

$a + b = 0 + 4 = 4$

중간값 정리 (Intermediate Value Theorem)

함수 $f$가 $[a, b]$에서 연속이고 $f(a) \neq f(b)$이면, $f(a)$와 $f(b)$ 사이의 임의의 값 $k$에 대하여

$$f(c) = k$$

를 만족하는 $c$가 $(a, b)$에 적어도 하나 존재한다.

따름 정리: $f(a) \cdot f(b) < 0$이면 $(a, b)$에서 $f(c) = 0$인 $c$가 존재 → 방정식의 실수 근 존재 증명에 활용.

예제 6. 중간값 정리로 실근 존재 증명

$f(x) = x^3 - 3x + 1$이 구간 $(1, 2)$에서 적어도 하나의 실근을 가짐을 보여라.

풀이

$f(1) = 1 - 3 + 1 = -1 < 0$

$f(2) = 8 - 6 + 1 = 3 > 0$

$f$는 연속이고 $f(1) \cdot f(2) < 0$이므로, 중간값 정리에 의해 $(1, 2)$에서 $f(c) = 0$인 $c$가 존재한다. $\blacksquare$

예제 7. 극한값 계산 — 차이 $\frac{\infty - \infty}$ 꼴

$\displaystyle\lim_{x \to \infty}(\sqrt{x^2 + 3x} - x)$를 구하여라.

풀이

유리화:

$= \displaystyle\lim_{x \to \infty}\dfrac{(\sqrt{x^2+3x}-x)(\sqrt{x^2+3x}+x)}{\sqrt{x^2+3x}+x} = \lim_{x \to \infty}\dfrac{3x}{\sqrt{x^2+3x}+x}$

$= \displaystyle\lim_{x \to \infty}\dfrac{3}{\sqrt{1+\frac{3}{x}}+1} = \dfrac{3}{1+1} = \dfrac{3}{2}$

예제 8. 좌·우 극한으로 극한 판별

$f(x) = \dfrac{|x-2|}{x-2}$에서 $\displaystyle\lim_{x \to 2}f(x)$의 존재 여부를 판별하여라.

풀이

$x > 2$: $|x-2| = x-2 \Rightarrow f(x) = 1$, 우극한 $= 1$

$x < 2$: $|x-2| = -(x-2) \Rightarrow f(x) = -1$, 좌극한 $= -1$

좌극한 $\neq$ 우극한이므로 $\displaystyle\lim_{x \to 2}f(x)$는 존재하지 않는다.

수능 핵심 전략

  • $\frac{0}{0}$ 꼴: 인수분해, 유리화, 로피탈(고교 과정에서는 비사용) — 인수 약분이 핵심
  • 연속 조건: 세 조건을 체크리스트처럼 확인 (정의됨, 극한 존재, 같음)
  • 불연속 분류: 제거 가능 불연속(점 불연속) vs. 점프 불연속 vs. 무한 불연속
  • 중간값 정리: "방정식의 실근이 존재한다"는 증명에 표준 도구

연습 문제

  1. $\displaystyle\lim_{x \to 1}\dfrac{x^2 - 1}{x^2 - 3x + 2}$를 구하여라.
  2. $\displaystyle\lim_{x \to 9}\dfrac{\sqrt{x} - 3}{x - 9}$를 구하여라.
  3. $\displaystyle\lim_{x \to \infty}\dfrac{2x^2 - 3x + 1}{x^2 + 5}$를 구하여라.
  4. $f(x) = \begin{cases}x^2 - 1 & (x \leq 1) \\ ax + b & (x > 1)\end{cases}$가 $x = 1$에서 연속이고 $f(3) = 7$일 때, $a + b$를 구하여라.
  5. $f(x) = 2x^3 - 3x - 1$이 $(1, 2)$에서 실근을 가짐을 중간값 정리로 보여라.
