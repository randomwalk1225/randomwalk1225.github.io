수학 I — 삼각함수

고등학교 2학년  |  2022 개정 교육과정  |  라디안 · sin·cos·tan · 그래프 · 삼각방정식 · 삼각부등식

삼각함수는 파동, 진동, 회전 등 주기 현상을 표현하는 도구입니다. 수능에서는 삼각함수 그래프의 변환, 삼각방정식·부등식이 매년 출제됩니다. 라디안과 도(°)의 변환, 삼각함수 사이의 관계식을 완벽히 익히는 것이 핵심입니다.

1. 각의 측정 — 라디안(radian)

정의: 라디안

반지름이 $r$인 원에서, 반지름과 같은 길이의 호에 대한 중심각의 크기를 1 라디안(radian, rad)이라 한다.

$$\pi \text{ rad} = 180°$$

변환 공식:

  • 도(°) → 라디안: 곱하기 $\dfrac{\pi}{180}$
  • 라디안 → 도(°): 곱하기 $\dfrac{180}{\pi}$

부채꼴 공식: 반지름 $r$, 중심각 $\theta$(라디안)일 때

$$l = r\theta, \quad S = \frac{1}{2}r^2\theta = \frac{1}{2}rl$$

예제 1. 라디안과 호도법

반지름 6, 중심각 $\dfrac{2\pi}{3}$인 부채꼴의 호의 길이와 넓이를 구하여라.

풀이

$l = 6 \times \dfrac{2\pi}{3} = 4\pi$

$S = \dfrac{1}{2} \times 6^2 \times \dfrac{2\pi}{3} = \dfrac{1}{2} \times 36 \times \dfrac{2\pi}{3} = 12\pi$

2. 삼각함수의 정의

단위원에서의 삼각함수 정의

좌표평면에서 원점을 중심으로 하는 단위원($r = 1$) 위의 점을 $P(x, y)$라 하고, $x$축 양의 방향에서 반시계 방향으로 잰 각도를 $\theta$라 하면:

$$\sin\theta = y, \quad \cos\theta = x, \quad \tan\theta = \frac{y}{x}\;(x \neq 0)$$

또한: $\csc\theta = \dfrac{1}{\sin\theta}$, $\sec\theta = \dfrac{1}{\cos\theta}$, $\cot\theta = \dfrac{1}{\tan\theta}$

삼각함수 사이의 관계

  • $\sin^2\theta + \cos^2\theta = 1$
  • $\tan\theta = \dfrac{\sin\theta}{\cos\theta}$
  • $1 + \tan^2\theta = \sec^2\theta$

주요 각도의 삼각함수 값:

$\theta$$0$$\dfrac{\pi}{6}$$\dfrac{\pi}{4}$$\dfrac{\pi}{3}$$\dfrac{\pi}{2}$
$\sin\theta$$0$$\dfrac{1}{2}$$\dfrac{\sqrt{2}}{2}$$\dfrac{\sqrt{3}}{2}$$1$
$\cos\theta$$1$$\dfrac{\sqrt{3}}{2}$$\dfrac{\sqrt{2}}{2}$$\dfrac{1}{2}$$0$
$\tan\theta$$0$$\dfrac{1}{\sqrt{3}}$$1$$\sqrt{3}$정의 안 됨

예제 2. 삼각함수 간의 변환

$\sin\theta + \cos\theta = \dfrac{1}{\sqrt{2}}$일 때, $\sin\theta \cdot \cos\theta$와 $\sin^3\theta + \cos^3\theta$를 구하여라.

풀이

양변을 제곱: $\sin^2\theta + 2\sin\theta\cos\theta + \cos^2\theta = \dfrac{1}{2}$

$1 + 2\sin\theta\cos\theta = \dfrac{1}{2} \Rightarrow \sin\theta\cos\theta = -\dfrac{1}{4}$

$\sin^3\theta + \cos^3\theta = (\sin\theta + \cos\theta)(1 - \sin\theta\cos\theta)$

$= \dfrac{1}{\sqrt{2}} \left(1 + \dfrac{1}{4}\right) = \dfrac{1}{\sqrt{2}} \cdot \dfrac{5}{4} = \dfrac{5}{4\sqrt{2}} = \dfrac{5\sqrt{2}}{8}$

3. 삼각함수의 그래프와 성질

$y = a\sin(bx + c) + d$의 성질

  • 진폭(amplitude): $|a|$
  • 주기(period): $\dfrac{2\pi}{|b|}$
  • 위상(phase) 이동: $-\dfrac{c}{b}$ (수평 이동)
  • 수직 이동: $d$

$y = a\cos(bx + c) + d$도 동일한 성질 적용.

$y = \tan(bx + c)$의 주기: $\dfrac{\pi}{|b|}$

예제 3. 삼각함수 그래프 분석

$y = 3\sin\!\left(2x - \dfrac{\pi}{3}\right) + 1$의 진폭, 주기, 최댓값, 최솟값을 구하여라.

풀이

진폭: $|3| = 3$

주기: $\dfrac{2\pi}{2} = \pi$

최댓값: $3 \cdot 1 + 1 = 4$

최솟값: $3 \cdot (-1) + 1 = -2$

삼각함수의 대칭·주기 성질

  • $\sin(-\theta) = -\sin\theta$ (기함수), $\cos(-\theta) = \cos\theta$ (우함수)
  • $\sin(\pi - \theta) = \sin\theta$, $\cos(\pi - \theta) = -\cos\theta$
  • $\sin\!\left(\dfrac{\pi}{2} - \theta\right) = \cos\theta$, $\cos\!\left(\dfrac{\pi}{2} - \theta\right) = \sin\theta$

4. 삼각방정식과 삼각부등식

예제 4. 삼각방정식

$0 \leq x < 2\pi$에서 $2\cos^2 x - \cos x - 1 = 0$을 풀어라.

풀이

$t = \cos x$로 치환: $2t^2 - t - 1 = 0$

$(2t + 1)(t - 1) = 0 \Rightarrow t = -\dfrac{1}{2}$ 또는 $t = 1$

$\cos x = 1 \Rightarrow x = 0$

$\cos x = -\dfrac{1}{2} \Rightarrow x = \dfrac{2\pi}{3}$ 또는 $x = \dfrac{4\pi}{3}$

따라서 $x = 0,\; \dfrac{2\pi}{3},\; \dfrac{4\pi}{3}$

예제 5. 삼각부등식

$0 \leq x < 2\pi$에서 $\sin x > \dfrac{\sqrt{3}}{2}$를 풀어라.

풀이

$\sin x = \dfrac{\sqrt{3}}{2}$인 $x$: $x = \dfrac{\pi}{3}, \dfrac{2\pi}{3}$

사인 함수는 $\dfrac{\pi}{3}$ ~ $\dfrac{2\pi}{3}$ 구간에서 $\dfrac{\sqrt{3}}{2}$ 이상:

$\dfrac{\pi}{3} < x < \dfrac{2\pi}{3}$

예제 6. 삼각함수 덧셈정리

$\sin 75°$의 값을 덧셈정리를 이용하여 구하여라.

풀이

$\sin 75° = \sin(45° + 30°) = \sin 45°\cos 30° + \cos 45°\sin 30°$

$= \dfrac{\sqrt{2}}{2} \cdot \dfrac{\sqrt{3}}{2} + \dfrac{\sqrt{2}}{2} \cdot \dfrac{1}{2} = \dfrac{\sqrt{6}}{4} + \dfrac{\sqrt{2}}{4} = \dfrac{\sqrt{6}+\sqrt{2}}{4}$

삼각함수 덧셈정리

  • $\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$
  • $\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$
  • $\tan(\alpha + \beta) = \dfrac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta}$

예제 7. 배각 공식 활용

$\sin\theta = \dfrac{3}{5}$이고 $\theta$가 제2사분면 각일 때, $\sin 2\theta$와 $\cos 2\theta$를 구하여라.

풀이

제2사분면이므로 $\cos\theta < 0$: $\cos\theta = -\dfrac{4}{5}$

$\sin 2\theta = 2\sin\theta\cos\theta = 2 \cdot \dfrac{3}{5} \cdot \left(-\dfrac{4}{5}\right) = -\dfrac{24}{25}$

$\cos 2\theta = \cos^2\theta - \sin^2\theta = \dfrac{16}{25} - \dfrac{9}{25} = \dfrac{7}{25}$

예제 8. 사인 법칙과 코사인 법칙

삼각형 ABC에서 $a = 7, b = 5, C = 60°$일 때, 변 $c$의 길이를 구하여라.

풀이

코사인 법칙: $c^2 = a^2 + b^2 - 2ab\cos C$

$= 49 + 25 - 2 \cdot 7 \cdot 5 \cdot \cos 60° = 74 - 70 \cdot \dfrac{1}{2} = 74 - 35 = 39$

$c = \sqrt{39}$

수능 핵심 전략

  • 단위원: 삼각함수의 부호, 사분면별 값을 단위원으로 시각화
  • 이차 치환: $\sin^2 + \cos^2 = 1$로 한 함수로 통일 후 이차방정식
  • 그래프 변환: $y = \sin(2x - \pi/3)$ → $y = \sin 2(x - \pi/6)$으로 변환 후 이동 파악
  • 덧셈정리 암기: 사인·코사인·탄젠트 덧셈정리는 반드시 암기

  2. $\tan\theta = 2$일 때, $\sin\theta\cos\theta$와 $\sin^2\theta - \cos^2\theta$를 구하여라.
  3. $0 \leq x < 2\pi$에서 $2\sin^2 x - 3\sin x + 1 = 0$을 풀어라.
  4. $0 \leq x < 2\pi$에서 $\cos x \leq -\dfrac{1}{2}$를 풀어라.
  5. 삼각형 ABC에서 $a = 3, b = 4, c = 5$일 때, $\sin A : \sin B : \sin C$를 구하여라.
