공통수학 2 — 도형의 방정식

고등학교 1학년  |  2022 개정 교육과정  |  평면좌표 · 직선의 방정식 · 원의 방정식 · 도형의 이동

도형의 방정식은 기하학적 도형을 대수적 방정식으로 표현하는 단원입니다. 평면 위의 점·직선·원을 수식으로 다루며, 이후 벡터·공간도형·미적분과 연결되는 필수 단원입니다. 수능에서도 매년 출제되는 고배점 단원입니다.

1. 평면좌표

기본 공식 모음

두 점 사이의 거리 $AB = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
내분점 $m:n$으로 내분: $\left(\dfrac{mx_2+nx_1}{m+n},\;\dfrac{my_2+ny_1}{m+n}\right)$
중점 $\left(\dfrac{x_1+x_2}{2},\;\dfrac{y_1+y_2}{2}\right)$
외분점 $m:n$으로 외분: $\left(\dfrac{mx_2-nx_1}{m-n},\;\dfrac{my_2-ny_1}{m-n}\right)$

예제 1: 두 점 사이의 거리

$A(1, -2)$, $B(4, 2)$에 대해 $\overline{AB}$를 구하여라.

$AB = \sqrt{(4-1)^2 + (2-(-2))^2} = \sqrt{9 + 16} = \sqrt{25} = \mathbf{5}$

예제 2: 내분점과 외분점

$A(-1, 2)$, $B(5, 8)$에 대해 선분 $AB$를 $2:1$로 내분하는 점과 외분하는 점을 구하여라.

내분점: $\left(\dfrac{2\cdot5+1\cdot(-1)}{3},\;\dfrac{2\cdot8+1\cdot2}{3}\right) = \left(\dfrac{9}{3},\;\dfrac{18}{3}\right) = (3, 6)$

외분점: $\left(\dfrac{2\cdot5-1\cdot(-1)}{2-1},\;\dfrac{2\cdot8-1\cdot2}{2-1}\right) = (11, 14)$

예제 3: 삼각형의 무게중심

$A(1, 3)$, $B(-2, 0)$, $C(5, 6)$으로 이루어진 삼각형의 무게중심을 구하여라.

무게중심 $= \left(\dfrac{x_A+x_B+x_C}{3},\;\dfrac{y_A+y_B+y_C}{3}\right) = \left(\dfrac{1-2+5}{3},\;\dfrac{3+0+6}{3}\right) = \mathbf{(4/3,\; 3)}$

2. 직선의 방정식

직선의 방정식 형태

형태식특징
기울기-절편형$y = mx + b$기울기 $m$, $y$절편 $b$
점-기울기형$y - y_1 = m(x - x_1)$점 $(x_1, y_1)$을 지나고 기울기 $m$
두 점을 지나는 직선$\dfrac{y-y_1}{y_2-y_1} = \dfrac{x-x_1}{x_2-x_1}$두 점 $(x_1,y_1)$, $(x_2,y_2)$를 지남
일반형$ax + by + c = 0$수직인 직선 포함 가능
절편형$\dfrac{x}{a} + \dfrac{y}{b} = 1$$x$절편 $a$, $y$절편 $b$

예제 4: 두 점을 지나는 직선

두 점 $A(1, 2)$, $B(3, 6)$을 지나는 직선의 방정식을 구하여라.

기울기 $= \dfrac{6-2}{3-1} = 2$

점-기울기형: $y - 2 = 2(x-1)$ → $y = 2x$

예제 5: 평행과 수직 조건

직선 $y = 3x + 1$에 평행한 직선과 수직인 직선의 기울기를 구하여라.

평행: 기울기 동일 → $m = 3$

수직: 기울기의 곱 $= -1$ → $m = -\dfrac{1}{3}$

점과 직선 사이의 거리

점 $(x_0, y_0)$에서 직선 $ax + by + c = 0$까지의 거리:

$$d = \frac{|ax_0 + by_0 + c|}{\sqrt{a^2 + b^2}}$$

예제 6: 점과 직선 사이의 거리

점 $(2, 1)$에서 직선 $3x - 4y + 5 = 0$까지의 거리를 구하여라.

$d = \dfrac{|3(2) - 4(1) + 5|}{\sqrt{9+16}} = \dfrac{|6-4+5|}{5} = \dfrac{7}{5}$

3. 원의 방정식

원의 방정식 기본형과 일반형

기본형: 중심 $(a, b)$, 반지름 $r$인 원:

$$(x-a)^2 + (y-b)^2 = r^2$$

일반형:

$$x^2 + y^2 + Dx + Ey + F = 0 \quad \Leftrightarrow \quad \text{기본형으로 변환 가능}$$

중심: $\left(-\dfrac{D}{2},\;-\dfrac{E}{2}\right)$, 반지름: $\sqrt{\dfrac{D^2+E^2}{4}-F}$

예제 7: 원의 방정식 세우기

중심이 $(2, -3)$이고 반지름이 5인 원의 방정식을 구하여라.

$(x-2)^2 + (y+3)^2 = 25$

예제 8: 일반형 → 기본형 변환

$x^2 + y^2 - 4x + 6y - 3 = 0$을 기본형으로 변환하여 중심과 반지름을 구하여라.

$(x^2 - 4x + 4) + (y^2 + 6y + 9) = 3 + 4 + 9$

$(x-2)^2 + (y+3)^2 = 16$

중심: $(2, -3)$, 반지름: $4$

3-1. 원과 직선의 위치 관계

원과 직선의 위치 관계

원의 중심에서 직선까지의 거리 $d$, 반지름 $r$일 때:

조건위치 관계교점 수
$d < r$직선이 원과 두 점에서 교차2개
$d = r$직선이 원에 접함 (접선)1개
$d > r$직선이 원과 만나지 않음0개

예제 9: 원과 직선의 교점 수 판별

원 $(x-1)^2 + (y-2)^2 = 25$와 직선 $3x + 4y - 10 = 0$의 위치 관계를 조사하여라.

중심 $(1, 2)$에서 직선까지의 거리:

$d = \dfrac{|3(1)+4(2)-10|}{5} = \dfrac{|3+8-10|}{5} = \dfrac{1}{5}$

$d = \dfrac{1}{5} < r = 5$ → 두 점에서 교차

예제 10: 접선의 방정식

원 $x^2 + y^2 = 10$ 위의 점 $(1, 3)$에서의 접선의 방정식을 구하여라.

원 $x^2 + y^2 = r^2$ 위의 점 $(x_0, y_0)$에서의 접선: $x_0 x + y_0 y = r^2$

따라서: $1 \cdot x + 3 \cdot y = 10$

$x + 3y = 10$

4. 도형의 이동

평행이동

점 $(x, y)$를 $x$축 방향으로 $m$, $y$축 방향으로 $n$만큼 평행이동:

$$(x, y) \;\longrightarrow\; (x+m,\; y+n)$$

도형의 방정식 $f(x, y) = 0$을 같은 방향으로 이동하면:

$$f(x-m,\; y-n) = 0$$

대칭이동

대칭축 / 대칭점점 $(x, y)$의 이동
$x$축 대칭$(x, -y)$
$y$축 대칭$(-x, y)$
원점 대칭$(-x, -y)$
직선 $y = x$ 대칭$(y, x)$

예제 11: 도형의 평행이동

직선 $y = 2x + 1$을 $x$축 방향으로 $3$, $y$축 방향으로 $-2$만큼 평행이동한 직선의 방정식을 구하여라.

$x$ 대신 $x - 3$, $y$ 대신 $y + 2$를 대입:

$y + 2 = 2(x-3) + 1$

$y + 2 = 2x - 5$

$y = 2x - 7$

예제 12: 원의 대칭이동

원 $(x-2)^2 + (y+1)^2 = 9$를 $x$축에 대칭이동한 원의 방정식을 구하여라.

$x$축 대칭: $y$ 대신 $-y$를 대입

$(x-2)^2 + (-y+1)^2 = 9$

$(x-2)^2 + (y-1)^2 = 9$

중심: $(2, 1)$, 반지름: $3$ (중심의 $y$좌표만 부호 변환)

연습 문제

  1. $A(3, -1)$, $B(-1, 5)$의 중점과 두 점 사이의 거리를 구하여라.
  2. 두 점 $(0, 3)$, $(4, 0)$을 지나는 직선의 방정식을 구하여라.
  3. 점 $(3, 4)$에서 직선 $4x - 3y + 1 = 0$까지의 거리를 구하여라.
  4. $x^2 + y^2 - 6x + 2y + 6 = 0$을 기본형으로 변환하여 중심과 반지름을 구하여라.
  5. 원 $x^2 + y^2 = 25$ 위의 점 $(3, 4)$에서의 접선의 방정식을 구하여라.

시험 대비 핵심 정리

  • 두 점 거리: $\sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$ — 반드시 암기
  • 점과 직선 거리: $\dfrac{|ax_0+by_0+c|}{\sqrt{a^2+b^2}}$ — 절댓값과 분모 혼동 주의
  • 원의 기본형 변환: 완전제곱식 이용 ($x^2 + bx = (x + b/2)^2 - (b/2)^2$)
  • 접선 공식: 원 위의 점 $(x_0, y_0)$에서 $x_0 x + y_0 y = r^2$
  • 평행이동: 방정식에서 $x \to x-m$, $y \to y-n$ (이동 방향과 반대 부호)
  • 내분점 공식: $m:n$으로 $\left(\dfrac{mx_2+nx_1}{m+n},\;\dfrac{my_2+ny_1}{m+n}\right)$
