로그함수의 미분과 적분

2025년 10월 19일 16:00
← 목록으로 돌아가기

1. 로그함수의 기초

로그함수는 지수함수의 역함수입니다. 자연로그는 밑이 $e$ (오일러 수, $e \approx 2.71828...$)인 로그를 의미하며 $\ln x$로 표기합니다. 일반로그는 밑이 $a$ (단, $a > 0$, $a \neq 1$)인 로그를 의미하며 $\log_a x$로 표기합니다.

중요:
  • 자연로그: $\ln x = \log_e x$ (밑이 $e$)
  • 상용로그: $\log x = \log_{10} x$ (밑이 10, 공학용 계산기에서 많이 사용)
  • 밑 변환 공식: $\log_a x = \frac{\ln x}{\ln a}$

2. 자연로그의 미분

자연로그의 미분 공식

기본 공식:

$$\frac{d}{dx}\ln x = \frac{1}{x} \quad (x > 0)$$

합성함수의 경우 (연쇄법칙):

$$\frac{d}{dx}\ln[f(x)] = \frac{f'(x)}{f(x)} = \frac{1}{f(x)} \cdot f'(x)$$

절댓값을 포함한 경우:

$$\frac{d}{dx}\ln|x| = \frac{1}{x} \quad (x \neq 0)$$

자연로그 미분 유도

$y = \ln x$라 하면 $x = e^y$입니다.

양변을 $x$에 대해 미분하면:

$$1 = e^y \cdot \frac{dy}{dx}$$

정리하면:

$$\frac{dy}{dx} = \frac{1}{e^y} = \frac{1}{x}$$

따라서:

$$\frac{d}{dx}\ln x = \frac{1}{x}$$

3. 일반로그의 미분

일반로그의 미분 공식

밑이 $a$인 로그:

$$\frac{d}{dx}\log_a x = \frac{1}{x \ln a} \quad (x > 0, a > 0, a \neq 1)$$

합성함수의 경우:

$$\frac{d}{dx}\log_a[f(x)] = \frac{f'(x)}{f(x) \ln a}$$

일반로그 미분 유도

밑 변환 공식을 사용합니다:

$$\log_a x = \frac{\ln x}{\ln a}$$

양변을 $x$에 대해 미분하면:

$$\frac{d}{dx}\log_a x = \frac{d}{dx}\left(\frac{\ln x}{\ln a}\right)$$

$\ln a$는 상수이므로:

$$= \frac{1}{\ln a} \cdot \frac{d}{dx}\ln x = \frac{1}{\ln a} \cdot \frac{1}{x}$$

따라서:

$$\frac{d}{dx}\log_a x = \frac{1}{x \ln a}$$

4. 로그함수의 적분

로그함수의 적분 공식

자연로그의 적분:

$$\int \ln x \, dx = x\ln x - x + C = x(\ln x - 1) + C$$

일반로그의 적분:

$$\int \log_a x \, dx = \frac{x}{\ln a}(\ln x - 1) + C$$

$\frac{1}{x}$의 적분:

$$\int \frac{1}{x} \, dx = \ln|x| + C$$

$\frac{f'(x)}{f(x)}$의 적분:

$$\int \frac{f'(x)}{f(x)} \, dx = \ln|f(x)| + C$$

$\ln x$의 적분 유도 (부분적분)

부분적분 공식 $\int u \, dv = uv - \int v \, du$를 사용합니다.

$u = \ln x$, $dv = dx$로 놓으면:

  • $du = \frac{1}{x} dx$
  • $v = x$

부분적분 공식에 대입하면:

$$\int \ln x \, dx = x \ln x - \int x \cdot \frac{1}{x} \, dx$$ $$= x \ln x - \int 1 \, dx$$ $$= x \ln x - x + C$$

5. 공식 요약표

함수 미분 적분
$\ln x$ $\frac{1}{x}$ $x\ln x - x + C$
$\log_a x$ $\frac{1}{x \ln a}$ $\frac{x}{\ln a}(\ln x - 1) + C$
$\frac{1}{x}$ $-\frac{1}{x^2}$ $\ln|x| + C$
$e^x$ $e^x$ $e^x + C$
$a^x$ $a^x \ln a$ $\frac{a^x}{\ln a} + C$

6. 연습문제

연습문제 1

다음 함수를 미분하시오:

(1) $f(x) = \ln(3x^2 + 1)$

(2) $g(x) = x^2 \ln x$

풀이

(1) $f(x) = \ln(3x^2 + 1)$의 미분

연쇄법칙을 사용합니다:

$$f'(x) = \frac{1}{3x^2 + 1} \cdot (3x^2 + 1)'$$ $$= \frac{1}{3x^2 + 1} \cdot 6x$$ $$= \frac{6x}{3x^2 + 1}$$

(2) $g(x) = x^2 \ln x$의 미분

곱의 미분법을 사용합니다:

$$g'(x) = (x^2)' \ln x + x^2 (\ln x)'$$ $$= 2x \ln x + x^2 \cdot \frac{1}{x}$$ $$= 2x \ln x + x$$ $$= x(2\ln x + 1)$$

답:
(1) $f'(x) = \frac{6x}{3x^2 + 1}$
(2) $g'(x) = x(2\ln x + 1)$

연습문제 2

다음 적분을 계산하시오:

(1) $\displaystyle\int_1^e \frac{1}{x} \, dx$

(2) $\displaystyle\int \frac{2x}{x^2 + 1} \, dx$

풀이

(1) $\displaystyle\int_1^e \frac{1}{x} \, dx$

부정적분: $\int \frac{1}{x} \, dx = \ln|x| + C$

$x > 0$이므로 절댓값 기호를 제거할 수 있습니다:

$$\int_1^e \frac{1}{x} \, dx = \left[\ln x\right]_1^e$$ $$= \ln e - \ln 1$$ $$= 1 - 0 = 1$$

(2) $\displaystyle\int \frac{2x}{x^2 + 1} \, dx$

$u = x^2 + 1$로 치환하면 $du = 2x \, dx$이므로:

$$\int \frac{2x}{x^2 + 1} \, dx = \int \frac{1}{u} \, du$$ $$= \ln|u| + C$$ $$= \ln|x^2 + 1| + C$$

$x^2 + 1 > 0$이므로 절댓값 기호를 제거할 수 있습니다:

$$= \ln(x^2 + 1) + C$$

답:
(1) $1$
(2) $\ln(x^2 + 1) + C$

7. 활용 및 응용

로그함수의 미분과 적분은 다음과 같은 분야에서 널리 사용됩니다:

  • 경제학: 탄력성 계산, 성장률 분석, 복리 계산
  • 물리학: 감쇠 현상, 방사성 붕괴, 음향학 (데시벨)
  • 정보이론: 엔트로피, 정보량 측정
  • 통계학: 로그 정규분포, 최대우도추정법
  • 화학: pH 계산, 반응속도론
  • 생물학: 인구 성장 모델, 세포 분열

특히 자연로그는 자연 현상에서 나타나는 성장과 감쇠를 설명하는 데 필수적이며, 미적분학에서 가장 중요한 함수 중 하나입니다.
