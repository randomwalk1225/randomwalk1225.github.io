D-5 마인드셋: 지금 뭘 해야 하나

SAT 수학 만점(800점)을 받으려면 44문항 중 실수 없이 거의 모두 맞혀야 합니다. 시험이 5일도 안 남은 지금, 새로운 개념을 처음부터 공부하는 건 시간 낭비입니다. 오히려 이미 알고 있는 개념에서 실수하지 않는 것이 훨씬 중요합니다.

디지털 SAT 수학은 두 개의 모듈로 구성됩니다. 첫 번째 모듈 성적에 따라 두 번째 모듈의 난이도가 결정됩니다. 따라서 앞 문제에서 실수하지 않는 것이 전체 점수에 직결됩니다.

D-5 핵심 원칙
  • 새 단원 공부 시작 ❌ → 약점 단원 1개만 집중 보완 ✅
  • 문제집 새로 사기 ❌ → 틀렸던 문제 오답 정리 ✅
  • 밤새워 공부 ❌ → 수면 확보 + 짧고 집중적인 복습 ✅

5일 플랜 — 하루별 전략

D-5 · 3월 9일(월)
🔎 약점 단원 1개 집중 드릴
  • 지금까지 가장 많이 틀렸던 단원 1개만 고른다 (이차함수, 연립방정식, 통계 중 하나)
  • 해당 단원 문제 15~20문항 풀기 — 타이머 없이 천천히, 개념 확인하며 풀기
  • 틀린 문제 오답 노트에 정리 (왜 틀렸는지 한 문장으로 쓰기)
  • 공식 카드 만들기 (아래 핵심 공식 섹션 참고)
D-4 · 3월 10일(화)
🔎 약점 단원 마무리 + 오답 복습
  • D-5에서 틀린 문제 다시 풀기 (오답 복습)
  • 같은 단원 추가 문제 10문항 — 이번엔 타이머 사용 (문항당 90초)
  • 이전에 봤던 모의고사 오답 전체 훑어보기 (새 문제 풀 필요 없음)
  • 저녁에는 공식 카드 한 번 더 보고 취침
D-3 · 3월 11일(수)

  • College Board 공식 Practice Test (Bluebook 앱 또는 PDF) 1회 전체 실전처럼 풀기
  • 수학 모듈 1: 35분 타이머, 계산기 허용, 외부 자료 없이
  • 수학 모듈 2: 35분 타이머, 동일 조건
  • 채점 후 틀린 문제 유형 분류 (계산 실수 / 개념 오해 / 시간 부족)
  • 풀이 후 많이 틀린 유형 한 번 더 확인

📚 오답 노트 최종 점검 + 수면 확보

  • 공식 카드 마지막으로 5분 훑기

반드시 외워야 할 핵심 공식

SAT 수학 시험지에는 일부 공식이 제공되지만, 아래 공식들은 별도 제공 없이 암기해야 하거나, 빠른 풀이를 위해 즉시 활용할 수 있어야 합니다.

기울기 (Slope)
$$m = \frac{y_2 - y_1}{x_2 - x_1}$$
이차방정식 근의 공식
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$
판별식 (Discriminant)
$$D = b^2 - 4ac$$

D>0: 두 실근   D=0: 중근   D<0: 허근

꼭짓점의 x좌표
$$x_{vertex} = -\frac{b}{2a}$$
퍼센트 변화율
$$\% \text{ change} = \frac{\text{New} - \text{Old}}{\text{Old}} \times 100$$
지수 성장/감소
$$f(t) = a \cdot (1 \pm r)^t$$

a: 초기값, r: 비율, t: 시간

추가로 확인해야 할 공식들

  • 평균(Mean): $\bar{x} = \dfrac{\sum x_i}{n}$
  • 이자 복리: $A = P\left(1 + \dfrac{r}{n}\right)^{nt}$
  • 직선의 방정식: $y = mx + b$ (기울기-절편), $y - y_1 = m(x - x_1)$ (점-기울기)
  • 절댓값 방정식: $|ax + b| = c$ → $ax + b = c$ 또는 $ax + b = -c$
  • 시스템 방정식: 대입법 또는 가감법 — 계산기로도 풀 수 있으니 활용할 것
💡 공식 암기 팁
근의 공식은 노래처럼 "마이너 비, 플러스 마이너스, 루트 비제곱 마이너스 사에이씨, 나누기 이에이"로 리듬감 있게 외우면 잊어버리지 않습니다. 30번 소리 내어 말하면 몸에 배입니다.

마지막 2일, 절대 하지 말아야 할 것

⚠️ D-2 ~ D-1에 이런 행동은 점수를 깎습니다
  • 새 문제집 풀기 — 처음 보는 문제를 틀리면 오히려 불안감이 커짐
  • 약한 단원 새로 공부하기 — 2일 만에 새 개념이 굳지 않음. 시간 낭비
  • 밤샘 공부 — 수면 부족은 집중력, 계산 속도, 실수율 모두에 직접 영향
  • 유튜브로 SAT "꿀팁" 영상 찾기 — 지금 필요한 건 새 정보가 아니라 안정감
  • 친구와 시험 난이도 예측 토론 — 불안만 증폭됨
  • 시험 전날 저녁 과식 — 소화 부담이 수면의 질을 낮춤

시험 당일 루틴

시험 당일은 새로운 결정을 최소화하고, 익숙한 루틴을 따르는 것이 최선입니다. 아래 체크리스트를 활용하세요.

✅ 시험 당일 체크리스트
  • ☑ 신분증 (여권 또는 학생증) 확인
  • ☑ 계산기 (허용 모델) + 여분 배터리
  • ☑ 연필 2개 이상 (Bluebook 앱 사용 시 노트북/태블릿 완충)
  • ☑ 간식 (바나나, 초콜릿, 물) — 쉬는 시간에 섭취
  • ☑ 시험장 30분 전 도착
  • ☑ 모르는 문제 → 마킹 후 넘어가기 (시간 낭비 금지)
  • ☑ 마지막 5분 — 빈칸 없이 모두 답 입력했는지 확인

문제 풀이 전략 — 시간 배분

SAT 수학 각 모듈은 22문항에 35분입니다. 평균 약 1분 35초 per 문항이지만, 쉬운 문제는 빠르게 넘기고 어려운 문제에 시간을 집중해야 합니다.

  • 1~10번 (쉬운 문제): 문항당 60~80초 목표 → 여유 시간 확보
  • 11~18번 (중간 난이도): 문항당 90초~2분
  • 19~22번 (어려운 문제): 남은 시간 투자, 그래도 모르면 일단 답 입력
  • 마지막 3분: 표시해둔 문제 재확인, 빈칸 없는지 체크
💡 어려운 문제 만났을 때
30초 이상 막히면 일단 넘어가세요. SAT 디지털은 뒤로 돌아올 수 있습니다. 모르는 문제에 집착하다 뒤에 쉬운 문제를 시간 부족으로 못 푸는 실수가 가장 흔한 점수 손실 원인입니다.
