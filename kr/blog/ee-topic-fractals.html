1. 프랙탈이란 무엇인가

프랙탈(fractal)은 1975년 수학자 브누아 만델브로(Benoit Mandelbrot)가 명명한 기하학적 대상으로, 어떤 스케일로 확대해도 전체와 유사한 구조가 반복되는 자기유사성(self-similarity)을 특징으로 합니다. 유클리드 기하학의 점(0차원), 선(1차원), 면(2차원), 공간(3차원)과 달리, 프랙탈은 정수가 아닌 비정수 차원을 가질 수 있습니다.

자연에서 프랙탈 구조는 놀라울 정도로 흔합니다. 해안선의 굴곡, 고사리 잎의 가지치기, 번개의 갈라짐, 혈관의 분기, 구름의 윤곽 — 모두 프랙탈적 특성을 보입니다. 만델브로가 던진 유명한 질문 "영국의 해안선 길이는 얼마인가?"는 프랙탈 기하학의 출발점입니다. 측정 자(ruler)의 길이를 줄일수록 해안선의 측정 길이는 계속 증가하며, 이론적으로는 무한대로 발산합니다.

IB EE에서 프랙탈은 시각적으로 매혹적이면서도 수학적으로 풍부한 주제입니다. 복소수, 반복(iteration), 차원 이론이라는 세 가지 수학 영역을 자연스럽게 통합할 수 있습니다.

연구 질문 예시: "만델브로 집합의 경계에서 반복 횟수에 따른 탈출 속도(escape speed)의 분포가 집합의 기하학적 구조에 대해 무엇을 알려주는가?"

2. 만델브로 집합의 정의

만델브로 집합(Mandelbrot set) $\mathcal{M}$은 복소 평면에서 다음 반복 관계의 궤도가 발산하지 않는 복소수 $c$의 집합으로 정의됩니다:

$$z_{n+1} = z_n^2 + c, \quad z_0 = 0$$

좀 더 형식적으로, $c \in \mathcal{M}$이면 수열 $\{z_n\}$이 유계(bounded)입니다. 즉 $|z_n|$이 무한대로 발산하지 않습니다. 수학적으로 $|z_n| > 2$가 되는 순간 수열은 반드시 발산하므로, 이 값을 탈출 반경(escape radius)으로 사용합니다.

예시:
$c = 0$: $z_1 = 0, z_2 = 0, z_3 = 0, \ldots$ → 유계 → $c = 0 \in \mathcal{M}$
$c = 1$: $z_1 = 1, z_2 = 2, z_3 = 5, z_4 = 26, \ldots$ → 발산 → $c = 1 \notin \mathcal{M}$
$c = -1$: $z_1 = -1, z_2 = 0, z_3 = -1, z_4 = 0, \ldots$ → 주기 2 → $c = -1 \in \mathcal{M}$
$c = i$: $z_1 = i, z_2 = -1+i, z_3 = -i, z_4 = -1+i, \ldots$ → 주기 2 → $c = i \in \mathcal{M}$

3. 만델브로 집합의 시각화

만델브로 집합을 시각화하는 알고리즘은 직관적입니다. 복소 평면의 각 점 $c$에 대해 반복을 수행하고, 발산까지의 반복 횟수에 따라 색상을 부여합니다. 집합에 속하는 점(발산하지 않는 점)은 보통 검은색으로 칠합니다.

Python으로 구현하면 약 20줄의 코드로 만델브로 집합을 그릴 수 있습니다. 복소 평면에서 실수부 $[-2.5, 1.0]$, 허수부 $[-1.25, 1.25]$ 범위를 격자(grid)로 나누고, 각 격자점에서 최대 반복 횟수(예: 100회)까지 반복을 수행합니다.

만델브로 집합의 경계를 확대하면, 놀랍게도 원래 전체 형상과 유사한 소형 만델브로 집합들이 무한히 나타납니다. 이것이 바로 자기유사성이며, 확대 배율을 $10^{14}$ 이상으로 올려도 새로운 구조가 계속 드러납니다. 이 성질은 집합의 경계가 프랙탈임을 보여줍니다.

4. 줄리아 집합과의 관계

만델브로 집합과 밀접한 관계에 있는 것이 줄리아 집합(Julia set)입니다. 줄리아 집합은 고정된 $c$에 대해 다양한 초기값 $z_0$로 반복을 수행했을 때, 궤도가 유계인 $z_0$의 집합입니다. 즉 만델브로 집합이 "매개변수 공간"에서의 집합이라면, 줄리아 집합은 "초기값 공간"에서의 집합입니다.

핵심적인 정리가 있습니다: $c \in \mathcal{M}$이면 대응하는 줄리아 집합 $J_c$는 연결(connected)되고, $c \notin \mathcal{M}$이면 $J_c$는 완전 불연결(totally disconnected, Cantor dust)됩니다. 이 정리를 시각적으로 보여주는 것은 EE에서 매우 효과적인 분석 소재입니다.

5. 프랙탈 차원

프랙탈의 가장 중요한 수학적 특성은 비정수 차원입니다. 프랙탈 차원을 정량화하는 방법 중 IB 수준에서 접근 가능한 것이 박스 카운팅 차원(box-counting dimension)입니다:

$$D = \lim_{\epsilon \to 0} \frac{\ln N(\epsilon)}{\ln(1/\epsilon)}$$

여기서 $N(\epsilon)$은 한 변의 길이가 $\epsilon$인 정사각형(박스)으로 프랙탈을 덮을 때 필요한 최소 박스 수입니다. $\epsilon$을 줄여가며 $N(\epsilon)$을 측정하고, $\ln N(\epsilon)$ 대 $\ln(1/\epsilon)$의 그래프를 그리면 기울기가 프랙탈 차원 $D$에 해당합니다.

고전적 프랙탈의 차원:
코흐 눈송이(Koch snowflake): $D = \frac{\ln 4}{\ln 3} \approx 1.262$
시어핀스키 삼각형(Sierpinski triangle): $D = \frac{\ln 3}{\ln 2} \approx 1.585$
만델브로 집합의 경계: $D = 2$ (경계가 놀랍도록 복잡)

6. EE에서의 탐구 방향

반복 횟수와 경계의 복잡도: 최대 반복 횟수를 10, 50, 100, 500, 1000으로 변화시키며 만델브로 집합의 시각화가 어떻게 변하는지 분석합니다. 반복 횟수가 증가할수록 경계의 세부 구조가 드러나며, 이를 통해 "무한 반복의 극한"이라는 수학적 개념을 직관적으로 이해할 수 있습니다.

줄리아 집합의 형태 탐구: 만델브로 집합의 다양한 위치에서 $c$를 선택하여 대응하는 줄리아 집합을 그리고, $c$의 위치(집합 내부, 경계, 외부)에 따른 줄리아 집합의 형태 변화를 분류합니다.

코흐 눈송이의 박스 카운팅 차원 측정: 실제로 격자를 사용하여 코흐 눈송이의 프랙탈 차원을 실험적으로 측정하고, 이론값 $\ln 4 / \ln 3$과 비교합니다. 측정의 오차 원인을 분석하는 것도 비판적 사고 요소입니다.
