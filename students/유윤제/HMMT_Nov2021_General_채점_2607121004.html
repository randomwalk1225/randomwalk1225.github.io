Q1 ✓ Correct Combinatorics · dominoes
Official\(4\) Yunje\(4\)
Problem
A domino has a left end and a right end, each of a certain color. Alice has four dominos, colored red-red, red-blue, blue-red, and blue-blue. Find the number of ways to arrange them in a row end-to-end so that adjacent ends have the same color. The dominos cannot be rotated.
Solution
  1. Each domino is a directed color pair; matching adjacent ends means the right color of one equals the left color of the next — an Euler-path count on the two-vertex \(\{R,B\}\) multigraph.
  2. WLOG the first domino's left end is \(R\). If it is \(RR\): forced \(RR,RB,BB,BR\) — one way. If it is \(RB\): forced \(RB,BB,BR,RR\) — one way.
  3. So \(2\) arrangements start with a red left end; by the \(R\leftrightarrow B\) symmetry, \(2\) start with blue.
  4. \(\therefore 2+2=4.\)
→ Official solution (P1)
Q2 ✗ Wrong Number theory · exponents
Official\(117\) Yunje\(13\)
Problem
Suppose \(a\) and \(b\) are positive integers for which \(8a^a b^b=27a^b b^a\). Find \(a^2+b^2\).
Solution
  1. Rearrange \(8a^a b^b=27a^b b^a\) to \(\dfrac{a^a b^b}{a^b b^a}=\dfrac{27}{8}\), i.e. \(\left(\dfrac{a}{b}\right)^{a-b}=\dfrac{27}{8}=\left(\dfrac32\right)^3.\)
  2. With \(a,b\) positive integers, match base and exponent: \(\dfrac{a}{b}=\dfrac32,\ a-b=3\Rightarrow a=9,\ b=6\) (the swap \(a=6,b=9\) gives the same pair). The \(27/8=(27/8)^1\) branches give no integers.
  3. So \(\{a,b\}=\{9,6\}.\)
  4. \(\therefore a^2+b^2=81+36=117.\)
→ Official solution (P2)
Q3 ✓ Correct Geometry · circle in a square
Official\(807\) Yunje\(807\)
Problem
Let \(ABCD\) be a unit square. A circle with radius \(\tfrac{32}{49}\) passes through \(D\) and is tangent to side \(AB\) at \(E\). Then \(DE=\tfrac{m}{n}\) with \(\gcd(m,n)=1\). Find \(100m+n\).
Solution
  1. Unit square \(ABCD\); circle center \(O\), radius \(r=\tfrac{32}{49}\), through \(D\) and tangent to \(AB\) at \(E\) (so \(OE\perp AB\)). Let \(F=OE\cap CD,\ x=DF.\)
  2. \(D\) on the circle: \(DF^2+OF^2=DO^2\Rightarrow x^2+(1-r)^2=r^2\Rightarrow x^2=2r-1.\)
  3. \(DE^2=DF^2+EF^2=x^2+1^2=1+(2r-1)=2r=\dfrac{64}{49}\Rightarrow DE=\dfrac{8}{7}.\)
  4. \(\therefore m/n=8/7,\ 100m+n=807.\)
→ Official solution (P3)
Q4 ✓ Correct Number theory · digit sums
Official\(11\) Yunje\(11\)
Problem
The sum of the digits of the time \(19\) minutes ago is two less than the sum of the digits of the time right now. Find the sum of the digits of the time in \(19\) minutes. (Standard \(12\)-hour clock \(hh{:}mm\).)
Solution
  1. On a \(12\)-hour clock, adding \(19\) minutes changes the digit sum by \(\equiv+1\pmod 9\) unless the hour rolls over — so 'two less' forces a rollover.
  2. That pins \(h=12\ (\to 1)\) with minutes \(m>40\); the exact relation \(4+s(m)=s(m-41)\) gives \(m=50\), so now \(=1{:}09.\)
  3. In \(19\) minutes the time is \(1{:}28.\)
  4. \(\therefore 1+2+8=11.\)
→ Official solution (P4)
Q5 ✓ Correct Probability · random chords
Official\(115\) Yunje\(115\)
Problem
A chord is drawn on a circle by choosing two points uniformly at random on its circumference; this is done three times total. The circle is cut along the three chords. The probability that one of the pieces is a triangle is \(\tfrac{m}{n}\) with \(\gcd(m,n)=1\). Find \(100m+n\).
Solution
  1. Pick \(6\) random points, then a random pairing into \(3\) chords. The three chords form a triangle iff each point is paired with its opposite among the six (so all three chords pairwise cross).
  2. Exactly one of the pairings achieves this. The number of ways to pair \(6\) points into \(3\) unordered pairs is \(\dfrac{6!}{2^3\,3!}=15.\)
  3. So the probability is \(\dfrac{1}{15}\Rightarrow m=1,\ n=15.\)
  4. The problem asks for \(100m+n\): \(100\cdot1+15=115.\) ✓ Yunje wrote \(115\) — correct.
→ Official solution (P5)
Q6 ✓ Correct Probability · expected value
Official\(1003\) Yunje\(1003\)
Problem
Mario has a deck of seven pairs of matching number cards and two pairs of matching Jokers (\(18\) cards). He draws from the top one by one until he holds a matching pair of Jokers. The expected number of complete pairs he then holds (including the Jokers) is \(\tfrac{m}{n}\), \(\gcd(m,n)=1\). Find \(100m+n\).
Solution
  1. Order the \(9\) pairs by the moment each is first completed — a uniform ordering. The count of complete pairs held equals the position of the first Joker-pair in this order.
  2. Equivalently, arrange \(7\) \(N\)'s and \(2\) \(J\)'s at random and find the expected position of the first \(J\).
  3. Each \(N\) is before both \(J\)'s with probability \(\tfrac13\Rightarrow\) expected \(N\)'s before the first \(J\) is \(\tfrac73\); expected position \(=\tfrac73+1=\tfrac{10}{3}.\)
  4. \(\therefore m/n=10/3,\ 100m+n=1003.\)
→ Official solution (P6)
Q7 ✗ Wrong Algebra · iterated function
Official\(932\) Yunje\(1011\)
Problem
Define \(f:\mathbb{R}\to\mathbb{R}\) by \(f(x)=\dfrac{1}{x^2+\sqrt{x^4+2x}}\) if \(x\notin(-\sqrt[3]{2},0]\), and \(f(x)=0\) otherwise. The sum of all real \(x\) with \(f^{10}(x)=1\) is \(\tfrac{a+b\sqrt c}{d}\) (\(d>0\), \(c\) square-free, \(\gcd(a,b,d)=1\)). Find \(1000a+100b+10c+d\).
Solution
  1. For \(x>0\), the definition forces \(x\,f(x)^2+x^2 f(x)=\tfrac12\), a relation symmetric in \(x\leftrightarrow f(x)\). Hence \(f\) is an involution: \(f(f(x))=x.\)
  2. So \(f^{10}(x)=1\Rightarrow f(x)=f^{11}(x)=f\!\big(f^{10}(x)\big)=f(1)\); the tower of \(10\) collapses to \(f(x)=f(1).\)
  3. That equation is the quadratic \(2(1+\sqrt3)x^2+2x-(1+\sqrt3)^2=0\), with roots \(x=1\) and \(x=-\tfrac{1+\sqrt3}{2}.\)
  4. Sum \(=1-\tfrac{1+\sqrt3}{2}=\tfrac{1-\sqrt3}{2}\Rightarrow a=1,b=-1,c=3,d=2,\ 1000a+100b+10c+d=932.\) See the deep dive below.
→ Official solution (P7)
Q8 ✓ Correct Combinatorics · non-crossing routes
Official\(8744\) Yunje\(8744\)
Problem
Eight points \(P_1,\dots,P_8\) lie clockwise on a circle. A route is a sequence of at least two points such that following them by straight segments never revisits a point or crosses the path. Find the number of routes.
Solution
  1. Fix a set of \(k\) points and require the route to use all of them without self-crossing: choose the start, then each new step must extend to a neighbor of the current visited arc \(\Rightarrow k\cdot 2^{k-2}\) routes.
  2. Sum over every subset of size \(k\ge2\): \(\displaystyle\sum_{k=2}^{8}\binom{8}{k}\,k\,2^{k-2}.\)
  3. \(=\binom88(8\cdot2^6)+\binom87(7\cdot2^5)+\cdots+\binom82(2\cdot2^0)=8744.\)
  4. \(\therefore 8744.\)
→ Official solution (P8)
Q9 ✗ Wrong Geometry · cyclic pentagon
Official\(2852\) Yunje\(9051\)
Problem
\(ABCDE\) is a cyclic convex pentagon with \(AC=BD=CE\). \(AC,BD\) meet at \(X\); \(BD,CE\) meet at \(Y\). If \(AX=6,\ XY=4,\ YE=7\), the area of \(ABCDE\) is \(\tfrac{a\sqrt b}{c}\) (\(c>0\), \(b\) square-free, \(\gcd(a,c)=1\)). Find \(100a+10b+c\).
Solution
  1. \(AC=BD\Rightarrow ABCD\) is an isosceles trapezoid; \(BD=CE\Rightarrow BCDE\) is too. These give \(CY=DY=DX-XY=AX-XY=2\) and \(BX=CX=3.\)
  2. Triangle \(CXY\) has sides \(3,2,4\); Heron gives \([CXY]=\tfrac34\sqrt{15}.\)
  3. Ratios along the diagonals: \([ABC]=[CDE]=\tfrac94[CXY]\) and \([ACE]=\tfrac{27}{2}[CXY]\), so \([ABCDE]=\big(\tfrac94+\tfrac94+\tfrac{27}{2}\big)[CXY]=18[CXY].\)
  4. \([ABCDE]=18\cdot\tfrac34\sqrt{15}=\tfrac{27}{2}\sqrt{15}\Rightarrow a=27,b=15,c=2,\ 100a+10b+c=2852.\)
→ Official solution (P9)
Q10 ✓ Correct Algebra · symmetric system
Official\(5272\) Yunje\(5272\)
Problem
Real numbers \(x,y,z\) satisfy \(x+xy+xyz=1,\ y+yz+xyz=2,\ z+xz+xyz=4\). The largest possible value of \(xyz\) is \(\tfrac{a+b\sqrt c}{d}\) (\(d>0\), \(c\) square-free, \(\gcd(a,b,d)=1\)). Find \(1000a+100b+10c+d\).
Solution
  1. Let \(p=xyz\) and \(q=(x+1)(y+1)(z+1)\). Factoring the three equations as \(x(1+y),\,y(1+z),\,z(1+x)\) and multiplying: \(pq=(1-p)(2-p)(4-p).\)
  2. Adding the three equations gives \(q-p=8-3p\Rightarrow q=8-2p.\)
  3. Substitute: \(p(8-2p)=(1-p)(2-p)(4-p)\Rightarrow(4-p)(p^2-5p+2)=0\Rightarrow p=4\ \text{or}\ \dfrac{5\pm\sqrt{17}}{2}.\)
  4. Largest is \(p=\dfrac{5+\sqrt{17}}{2}\Rightarrow a=5,b=1,c=17,d=2,\ 1000a+100b+10c+d=5272.\)
→ Official solution (P10)

Deep Dive — Problem 7: iterated functions, involutions, a collapsing tower

  1. Why the tower looks scary: \(f^{10}(x)=1\) stacks \(f\) ten deep — hopeless head-on. The move is to find structure in \(f\) itself.
  2. Spot the involution: for \(x>0\), the definition rearranges to \(x\,f(x)^2+x^2 f(x)=\tfrac12\), a relation unchanged when you swap \(x\leftrightarrow f(x)\). So \(f\) is its own inverse — an involution: \(f(f(x))=x.\)
  3. The tower collapses: with \(f\circ f=\text{id}\), \(f^{11}(x)=f(x)\). Since \(f^{10}(x)=1\), \(f(x)=f^{11}(x)=f\!\big(f^{10}(x)\big)=f(1)\). The whole problem shrinks to \(f(x)=f(1)\).
  4. Solve once: \(f(x)=f(1)\) is the quadratic \(2(1+\sqrt3)x^2+2x-(1+\sqrt3)^2=0\); one root is \(x=1\), and Vieta gives root-sum \(-\dfrac{2}{2(1+\sqrt3)}=-\dfrac{1}{1+\sqrt3}=\dfrac{1-\sqrt3}{2}.\) \(\therefore\) answer \(\mathbf{932}.\)
Practice 1. A function \(g\) satisfies \(g(g(x))=x\) for all real \(x\), and \(g(3)=5\). Compute \(g^{100}(3)\) and \(g^{101}(3)\).
Show solution
  1. \(g(g(x))=x\) means an even number of iterations is the identity: \(g^{2k}(x)=x.\)
  2. An odd number equals one application: \(g^{2k+1}(x)=g(x).\)
  3. \(\therefore g^{100}(3)=3,\quad g^{101}(3)=g(3)=5.\)
Practice 2. Without finding the roots individually, compute the sum of the two roots of \(2(1+\sqrt3)x^2+2x-(1+\sqrt3)^2=0\).
Show solution
  1. Vieta: sum \(=-\dfrac{2}{2(1+\sqrt3)}=-\dfrac{1}{1+\sqrt3}.\)
  2. Rationalize: \(-\dfrac{1}{1+\sqrt3}\cdot\dfrac{\sqrt3-1}{\sqrt3-1}=-\dfrac{\sqrt3-1}{2}\) (since \((1+\sqrt3)(\sqrt3-1)=2\)).
  3. \(\therefore \dfrac{1-\sqrt3}{2}\) — the answer without ever isolating a root.

Q1 ✓ 정답 조합 · 도미노
공식 정답\(4\) 유윤제\(4\)
문제
도미노는 왼쪽 끝과 오른쪽 끝이 각각 색을 가진다. Alice는 red-red, red-blue, blue-red, blue-blue의 네 도미노를 가진다. 이웃한 끝의 색이 같도록 한 줄로 이어 놓는 방법의 수를 구하라. 도미노는 회전할 수 없다.
풀이
  1. 각 도미노는 방향이 있는 색쌍. 이웃한 끝의 색이 같아야 함 = 앞 도미노의 오른쪽 색 \(=\) 뒤 도미노의 왼쪽 색 → 두 꼭짓점 \(\{R,B\}\) 멀티그래프의 오일러 경로 세기.
  2. WLOG 첫 도미노의 왼쪽 끝이 \(R\). \(RR\)이면 \(RR,RB,BB,BR\)로 강제 — \(1\)가지. \(RB\)이면 \(RB,BB,BR,RR\)로 강제 — \(1\)가지.
  3. 즉 왼쪽 끝이 빨강인 배열이 \(2\)가지; \(R\leftrightarrow B\) 대칭으로 파랑도 \(2\)가지.
  4. \(\therefore 2+2=4.\)
→ 공식 해설 (문제 1)
Q2 ✗ 오답 정수론 · 지수
공식 정답\(117\) 유윤제\(13\)
문제
양의 정수 \(a,b\)가 \(8a^a b^b=27a^b b^a\)를 만족한다. \(a^2+b^2\)을 구하라.
풀이
  1. \(8a^a b^b=27a^b b^a\)를 \(\dfrac{a^a b^b}{a^b b^a}=\dfrac{27}{8}\)로 정리 → \(\left(\dfrac{a}{b}\right)^{a-b}=\dfrac{27}{8}=\left(\dfrac32\right)^3.\)
  2. \(a,b\)가 양의 정수이므로 밑과 지수를 맞추면 \(\dfrac{a}{b}=\dfrac32,\ a-b=3\Rightarrow a=9,\ b=6\) (\(a=6,b=9\)은 같은 쌍). \((27/8)^1\) 분기는 정수해 없음.
  3. 즉 \(\{a,b\}=\{9,6\}.\)
  4. \(\therefore a^2+b^2=81+36=117.\)

Q5 ✓ 정답 확률 · 무작위 현
공식 정답\(115\) 유윤제\(115\)
문제
원 둘레에서 두 점을 균등하게 무작위로 골라 현을 그리는 일을 총 세 번 한다. 원을 세 현을 따라 자른다. 조각 중 하나가 삼각형일 확률이 \(\tfrac{m}{n}\ (\gcd(m,n)=1)\)일 때 \(100m+n\)을 구하라.
풀이
  1. \(6\)개 점을 무작위로 찍고 \(3\)개 현으로 무작위 짝짓기. 세 현이 삼각형을 이룰 필요충분조건은 각 점이 여섯 점 중 맞은편 점과 짝지어지는 것(세 현이 서로 교차).
  2. 이를 만족하는 짝짓기는 정확히 \(1\)가지. \(6\)점을 \(3\)개의 (순서 없는) 쌍으로 나누는 방법은 \(\dfrac{6!}{2^3\,3!}=15.\)
  3. 따라서 확률은 \(\dfrac{1}{15}\Rightarrow m=1,\ n=15.\)
  4. 문제는 \(100m+n\)을 요구: \(100\cdot1+15=115.\) ✓ 유윤제도 \(115\)로 정확히 제출 — 정답.

Q6 ✓ 정답 확률 · 기댓값
공식 정답\(1003\) 유윤제\(1003\)
문제
Mario는 일곱 쌍의 숫자 카드와 두 쌍의 조커(총 \(18\)장)를 가진다. 위에서 한 장씩 뽑아 조커 한 쌍이 손에 들어올 때까지 뽑는다. 그때 손에 든 완성된 쌍의 수(조커 포함)의 기댓값이 \(\tfrac{m}{n}\ (\gcd(m,n)=1)\)일 때 \(100m+n\)을 구하라.
풀이
  1. \(9\)개 쌍을 '처음 완성되는 순간' 순으로 배열 — 균등한 순서. 손에 든 완성 쌍 수 \(=\) 이 순서에서 조커쌍이 처음 나오는 위치.
  2. 동치로, \(7\)개의 \(N\)과 \(2\)개의 \(J\)를 무작위 배열하고 첫 \(J\)의 기대 위치를 구함.
  3. 각 \(N\)이 두 \(J\)보다 앞설 확률 \(\tfrac13\Rightarrow\) 첫 \(J\) 앞의 \(N\) 기댓값 \(\tfrac73\); 기대 위치 \(=\tfrac73+1=\tfrac{10}{3}.\)
  4. \(\therefore m/n=10/3,\ 100m+n=1003.\)
→ 공식 해설 (문제 6)
Q7 ✗ 오답 대수 · 반복 함수
공식 정답\(932\) 유윤제\(1011\)
문제
\(f:\mathbb{R}\to\mathbb{R}\)를 \(x\notin(-\sqrt[3]{2},0]\)이면 \(f(x)=\dfrac{1}{x^2+\sqrt{x^4+2x}}\), 그 외엔 \(f(x)=0\)으로 정의한다. \(f^{10}(x)=1\)인 모든 실수 \(x\)의 합은 \(\tfrac{a+b\sqrt c}{d}\ (d>0,\ c\ \text{square-free},\ \gcd(a,b,d)=1)\). \(1000a+100b+10c+d\)를 구하라.
풀이
  1. \(x>0\)에서 정의로부터 \(x\,f(x)^2+x^2 f(x)=\tfrac12\)이 나오고, 이 식은 \(x\leftrightarrow f(x)\)에 대해 대칭 → \(f\)는 involution: \(f(f(x))=x.\)
  2. 따라서 \(f^{10}(x)=1\Rightarrow f(x)=f^{11}(x)=f\!\big(f^{10}(x)\big)=f(1)\); \(10\)번 반복이 \(f(x)=f(1)\) 하나로 붕괴.
  3. 이 식은 이차식 \(2(1+\sqrt3)x^2+2x-(1+\sqrt3)^2=0\), 근은 \(x=1\)과 \(x=-\tfrac{1+\sqrt3}{2}.\)
  4. 합 \(=1-\tfrac{1+\sqrt3}{2}=\tfrac{1-\sqrt3}{2}\Rightarrow a=1,b=-1,c=3,d=2,\ 1000a+100b+10c+d=932.\) 아래 심화 해설 참고.
→ 공식 해설 (문제 7)
Q8 ✓ 정답 조합 · 비교차 경로
공식 정답\(8744\) 유윤제\(8744\)
문제
여덟 점 \(P_1,\dots,P_8\)이 원 위에 시계방향으로 놓여 있다. 경로란 두 개 이상의 점의 나열로, 직선 선분으로 이어 방문할 때 점을 두 번 방문하지 않고 경로가 자기 자신과 교차하지 않는 것이다. 경로의 개수를 구하라.
풀이
  1. \(k\)개 점 집합을 고정하고 그 전부를 자기교차 없이 쓰는 경로: 시작점을 고른 뒤, 매 단계는 지금까지 방문한 호(arc)의 이웃으로만 확장 \(\Rightarrow k\cdot 2^{k-2}\)가지.
  2. 크기 \(k\ge2\)인 모든 부분집합에 대해 합: \(\displaystyle\sum_{k=2}^{8}\binom{8}{k}\,k\,2^{k-2}.\)
  3. \(=\binom88(8\cdot2^6)+\binom87(7\cdot2^5)+\cdots+\binom82(2\cdot2^0)=8744.\)
  4. \(\therefore 8744.\)
→ 공식 해설 (문제 8)
Q9 ✗ 오답 기하 · 원에 내접한 오각형
공식 정답\(2852\) 유윤제\(9051\)
문제
\(ABCDE\)는 원에 내접한 볼록오각형으로 \(AC=BD=CE\). \(AC,BD\)는 \(X\)에서, \(BD,CE\)는 \(Y\)에서 만난다. \(AX=6,\ XY=4,\ YE=7\)일 때 넓이는 \(\tfrac{a\sqrt b}{c}\ (c>0,\ b\ \text{square-free},\ \gcd(a,c)=1)\). \(100a+10b+c\)를 구하라.
풀이
  1. \(AC=BD\Rightarrow ABCD\)는 등변사다리꼴; \(BD=CE\Rightarrow BCDE\)도 등변사다리꼴. 이로부터 \(CY=DY=DX-XY=AX-XY=2,\ BX=CX=3.\)
  2. 삼각형 \(CXY\)의 세 변은 \(3,2,4\); 헤론 공식으로 \([CXY]=\tfrac34\sqrt{15}.\)
  3. 대각선 위 비율: \([ABC]=[CDE]=\tfrac94[CXY],\ [ACE]=\tfrac{27}{2}[CXY]\) → \([ABCDE]=\big(\tfrac94+\tfrac94+\tfrac{27}{2}\big)[CXY]=18[CXY].\)
  4. \([ABCDE]=18\cdot\tfrac34\sqrt{15}=\tfrac{27}{2}\sqrt{15}\Rightarrow a=27,b=15,c=2,\ 100a+10b+c=2852.\)
→ 공식 해설 (문제 9)
Q10 ✓ 정답 대수 · 대칭 연립방정식
공식 정답\(5272\) 유윤제\(5272\)
문제
실수 \(x,y,z\)가 \(x+xy+xyz=1,\ y+yz+xyz=2,\ z+xz+xyz=4\)를 만족한다. \(xyz\)의 최댓값은 \(\tfrac{a+b\sqrt c}{d}\ (d>0,\ c\ \text{square-free},\ \gcd(a,b,d)=1)\). \(1000a+100b+10c+d\)를 구하라.
풀이
  1. \(p=xyz,\ q=(x+1)(y+1)(z+1)\). 세 식을 \(x(1+y),\,y(1+z),\,z(1+x)\)로 인수분해해 곱하면 \(pq=(1-p)(2-p)(4-p).\)
  2. 세 식을 더하면 \(q-p=8-3p\Rightarrow q=8-2p.\)
  3. 대입: \(p(8-2p)=(1-p)(2-p)(4-p)\Rightarrow(4-p)(p^2-5p+2)=0\Rightarrow p=4\ \text{또는}\ \dfrac{5\pm\sqrt{17}}{2}.\)
  4. 최댓값은 \(p=\dfrac{5+\sqrt{17}}{2}\Rightarrow a=5,b=1,c=17,d=2,\ 1000a+100b+10c+d=5272.\)
→ 공식 해설 (문제 10)

심화 — 7번: 반복 함수 · involution · 붕괴하는 반복탑

  1. 반복탑이 무섭게 보이는 이유: \(f^{10}(x)=1\)은 \(f\)를 열 번 겹쳐야 해서 직접 풀기 벅차다. 핵심은 \(f\) 자체의 구조를 찾는 것.
  2. involution 발견: \(x>0\)에서 정의를 정리하면 \(x\,f(x)^2+x^2 f(x)=\tfrac12\)이 나온다. 이 식은 \(x\)와 \(f(x)\)를 맞바꿔도 똑같다 → \(f\)는 자기 자신이 역함수인 involution: \(f(f(x))=x.\)
  3. 탑이 붕괴: \(f\circ f=\text{id}\)이면 \(f^{11}(x)=f(x)\). 그런데 \(f^{10}(x)=1\)이므로 \(f(x)=f^{11}(x)=f\!\big(f^{10}(x)\big)=f(1)\). 즉 문제가 \(f(x)=f(1)\) 하나로 줄어든다.
  4. 한 번만 풀기: \(f(x)=f(1)\)은 이차식 \(2(1+\sqrt3)x^2+2x-(1+\sqrt3)^2=0.\) 한 근이 \(x=1\)(자명), Vieta로 근의 합 \(=-\dfrac{2}{2(1+\sqrt3)}=-\dfrac{1}{1+\sqrt3}=\dfrac{1-\sqrt3}{2}.\) \(\therefore\) 답 \(\mathbf{932}.\)
연습문제 1. 함수 \(g\)가 모든 실수에서 \(g(g(x))=x\)를 만족하고 \(g(3)=5\)이다. \(g^{100}(3)\)과 \(g^{101}(3)\)을 구하라.
해설 보기
  1. \(g(g(x))=x\)이면 짝수 번 반복은 항등: \(g^{2k}(x)=x.\)
  2. 홀수 번 반복은 \(g\) 한 번과 같음: \(g^{2k+1}(x)=g(x).\)
  3. \(\therefore g^{100}(3)=3,\quad g^{101}(3)=g(3)=5.\)
연습문제 2. 근을 각각 구하지 말고, \(2(1+\sqrt3)x^2+2x-(1+\sqrt3)^2=0\)의 두 근의 합을 구하라.
해설 보기
  1. Vieta: 근의 합 \(=-\dfrac{2}{2(1+\sqrt3)}=-\dfrac{1}{1+\sqrt3}.\)
  2. 유리화: \(-\dfrac{1}{1+\sqrt3}\cdot\dfrac{\sqrt3-1}{\sqrt3-1}=-\dfrac{\sqrt3-1}{2}\) (분모 \((1+\sqrt3)(\sqrt3-1)=2\)).
  3. \(\therefore \dfrac{1-\sqrt3}{2}\) — 근을 직접 구하지 않고도 답이 나온다.
