# v2 / 02 — 검색 노출 계획 (AdSense 없이도 가능)

> 목표: AdSense 승인과 무관하게 **구글 검색에서 트래픽**을 만든다.
> 사실 이게 AdSense 승인의 전제조건이기도 하다 — 같은 작업.

## 현재 상태 (실측, 2026-06-12)

- ✅ 사이트 색인됨: `site:mathhub.io` 에 페이지 노출 확인. 블로그글(예: `/us/blog/sat-2026-schedule`)은 검색에 잡힘.
- ✅ robots.txt / sitemap.xml(309 URL) 정상.
- ✅ Cloudflare Pages 라이브, HTTPS.
- ❌ 분석 페이지들은 노출돼도 클릭/체류 유발 못 함 (풀이가 없어서).

→ 인프라는 문제 없다. **랭크할 만한 콘텐츠가 없는 게** 문제.

## 단계별 실행

### 1단계 — 측정 기반 마련 (먼저)
- Google Search Console 에서 실제 데이터 확인: 어떤 쿼리로 노출(impression)되는가? 클릭률(CTR)이 0에 가까운 페이지는? 평균 게재순위는?
- 이게 "지어낸 통계" 대신 쓸 **진짜 데이터**다. 어디에 수요가 있는지 여기서 나온다.

### 2단계 — 이길 수 있는 롱테일 타게팅
- 광역 키워드("SAT math") = Khan/CollegeBoard 와 경쟁 불가. 버린다.
- 롱테일("how to solve 2022 ap calc ab frq 5 differential equation") = 의도 명확·경쟁 약함. **여기에 진짜 풀이로 답한다.**
- 1페이지에 1문항 의도를 깊게. 검색의도와 1:1.

### 3단계 — 구조화 데이터(정직하게)
- 풀이 페이지에 `Article` + `author` schema. Q&A 형식이면 합법 범위에서 `QAPage`.
- ⚠️ 가짜 별점·FAQ 남발 금지(또 다른 정책 위반). 실제 있는 것만 마크업.

### 4단계 — 내부 링크 & 토픽 클러스터
- 풀이(문제) ↔ 교재 챕터(개념) 양방향 링크. "이 함정은 챕터 5에서" 식 자연 연결.
- 한 시험을 허브로, 문항 풀이를 스포크로 묶으면 주제 권위가 쌓인다.

### 5단계 — 콘텐츠 위생 정리 (랭킹 희석 제거)
- 풀이 없는 얇은 분석 페이지를 `noindex` → 사이트 평균 품질이 올라가 전체 랭킹에 도움(Helpful Content 는 사이트 단위 평가).
- 죽은 페이지·중복 정리.

## AdSense 와의 관계

이 계획을 8~12주 돌려서 **GSC 에 실제 클릭이 꾸준히 잡히기 시작하면** 그때 AdSense 재신청. 트래픽 있는 사이트는 거의 자동 승인된다. 트래픽 없이 신청 → 거절의 반복이 v1 이었다.

## 측정 지표 (이걸로 성공 판정)

| 지표 | 출처 | 목표(12주) |
|------|------|-----------|
| 색인된 풀이 페이지 수 | GSC | 20~30 (질 우선) |
| 검색 노출 쿼리 수 | GSC | 우상향 |
| 클릭(실트래픽) | GSC | 0 → 꾸준한 일간 클릭 |
| 평균 게재순위 | GSC | 상승 |

지표가 오르면 v3 없이 v2 유지. 막히면 사유 기록 후 v3.
