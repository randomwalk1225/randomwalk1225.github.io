# v2 / 01 — 신규 전략: 좁고 깊게, 진짜 풀이

> 원칙: AdSense 를 직접 노리지 않는다. **검색에서 진짜 유용해서 사람이 찾아오는 페이지**를 만든다.
> 그 결과로 트래픽이 생기면 AdSense 는 따라온다. 순서를 거꾸로 했던 게 v1 의 실패.

## 전략 전환 요약

| 축 | v1 (실패) | v2 (신규) |
|----|-----------|-----------|
| 콘텐츠 | 출처 없는 "리서치 분석" | **실제 문제 + 단계별 풀이** (검증 가능) |
| 양 | 425개 얇은 템플릿 | 소수 정예 (먼저 20~30개 탁월하게) |
| 범위 | 5개 시장 동시 | **1개 시장 1개 시험에 집중** (예: US SAT Math) |
| 권위 | 가짜 통계 | 직접 푼 풀이 = 실증된 전문성(E-E-A-T) |
| 목표 | AdSense 승인 | 검색 트래픽 → (부산물) AdSense |

## 이 사이트만 줄 수 있는 진짜 가치

운영자가 수학을 실제로 풀 수 있다. 이게 핵심 자산이다.
- College Board/Khan 은 "공식 정답"은 주지만, **무료로 + 한 문제를 끝까지 + 함정까지 설명한 단계별 풀이**는 의외로 검색에서 비어 있다.
- "2023 march sat no calculator question 18 solution" 같은 **롱테일**은 경쟁이 약하고 의도가 명확하다.
- 이건 지어낼 필요가 없다. 풀면 된다. = 독창적·검증가능·유용 3박자.

## 즉시 조치 (콘텐츠 출혈 멈추기)

가짜 리서치 페이지가 살아 있는 한 신청은 계속 거절된다. 둘 중 하나:

- **A. 정리(권장 1순위):** 풀이 없는 "analysis" 페이지를 `noindex` + sitemap 제거하거나, 진짜 풀이로 교체하기 전까지 비공개. 손해 없음(트래픽 0).
- **B. 교체:** 대표 시험 1개(예: 가장 최근 공개된 SAT/AP)를 골라 **실제 문제 풀이 페이지**로 다시 씀.

## 파일럿 1개 페이지 기준 (이게 통과하면 복제)

1. 실제 공개 문제 1세트를 **한 문항씩** 다룬다.
2. 각 문항: 문제 → 접근 → **단계별 풀이(MathJax 수식 실재)** → 흔한 실수 → 정답.
3. 지어낸 통계 0. 출처 있는 사실만(College Board 공개 자료는 링크).
4. 운영자 명의·날짜·"직접 작성" 명시.
5. 관련 교재 챕터로 내부 링크.

→ 이 1개가 검색에 잡히고 체류시간이 나오는지 본 뒤 확장. (헌법: 검증 우선, UI/대량화는 품질 확인 후)
