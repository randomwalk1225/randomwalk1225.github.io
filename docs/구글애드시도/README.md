## 핵심 교훈 (절대 회귀 금지)

1. **AdSense 거절은 기술 문제가 아니라 콘텐츠 품질 문제였다.**
   v1 에서 21+8건의 기술 수정을 모두 했는데도 거절됐다. 원인이 거기 없었기 때문.
2. **"분석 페이지"가 곧 원인이다.** 수학 분석 페이지에 실제 풀이 수식이 0개,
   대신 출처 없는 통계("평균 520점", "곡선이 더 관대")로 채운 가짜 리서치 = Google "가치 없는 콘텐츠 / 대량생성(scaled content abuse)" 정책 직격탄.
3. **양 < 질.** 425개 템플릿 페이지가 도움이 아니라 해가 된다.
4. **검색 노출과 AdSense 승인은 같은 문제다.** 실제 트래픽 있는 사이트가 승인된다.
   따라서 v2 는 AdSense 를 직접 노리지 않고 "검색에서 실제로 유용해서 사람이 찾아오는 사이트"를 먼저 만든다.

## 회귀 금지 목록 (다시 하지 말 것)

- ❌ 출처·근거 없는 "research / empirical study / our analysis" 프레이밍
- ❌ 지어낸 통계·점수곡선·정답률을 사실처럼 제시
- ❌ 새 시장(국가)·새 템플릿 페이지 대량 추가로 "콘텐츠 많아 보이게" 만들기
- ❌ favicon·sitemap 날짜 분산 같은 기술 체크리스트를 "이번엔 될 것"의 근거로 삼기
