Chapter 1: Limits and Continuity

AP Calculus AB / BC · Updated February 2026 · 20 min read

Limits are the foundation upon which all of calculus is built. Before you can understand derivatives or integrals, you need a solid grasp of what it means for a function to approach a value. In this chapter, we develop the concept of a limit from intuition to formal definition, explore techniques for evaluating limits, and then use limits to define what it means for a function to be continuous.

1.1 Introduction to Limits

Consider the function $f(x) = \dfrac{x^2 - 1}{x - 1}$. If you try to evaluate $f(1)$ directly, you get $\frac{0}{0}$, which is undefined. But what happens to the values of $f(x)$ as $x$ gets closer and closer to $1$?

If we plug in values near $1$, something interesting emerges:

  • $f(0.9) = 1.9$
  • $f(0.99) = 1.99$
  • $f(0.999) = 1.999$
  • $f(1.001) = 2.001$
  • $f(1.01) = 2.01$
  • $f(1.1) = 2.1$

Even though $f(1)$ is undefined, the output values clearly approach $2$ as $x$ approaches $1$. This is the core idea of a limit: we care about what value the function is heading toward, not necessarily what value (if any) the function actually takes.

Definition: Limit of a Function

We write

$$\lim_{x \to a} f(x) = L$$

and say "the limit of $f(x)$ as $x$ approaches $a$ equals $L$" if we can make the values of $f(x)$ arbitrarily close to $L$ by taking $x$ sufficiently close to $a$ (but not equal to $a$).

A crucial point: the limit depends on what happens near $a$, not at $a$. The function does not even need to be defined at $x = a$ for the limit to exist.

Explore limits: observe how f(x) = sin(x)/x approaches 1 as x → 0, even though f(0) is undefined.

One-Sided Limits

Sometimes a function approaches different values depending on whether $x$ approaches from the left or from the right. We distinguish between these with one-sided limits.

Definition: One-Sided Limits

Left-hand limit: $\displaystyle\lim_{x \to a^-} f(x) = L$ means $f(x) \to L$ as $x$ approaches $a$ from values less than $a$.

Right-hand limit: $\displaystyle\lim_{x \to a^+} f(x) = L$ means $f(x) \to L$ as $x$ approaches $a$ from values greater than $a$.

The two-sided limit $\displaystyle\lim_{x \to a} f(x) = L$ exists if and only if both one-sided limits exist and are equal:

$$\lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = L$$

Worked Example 1.1

Consider the piecewise function

$$g(x) = \begin{cases} x + 1, & x < 2 \\ 5, & x = 2 \\ 7 - x, & x > 2 \end{cases}$$

Find $\displaystyle\lim_{x \to 2} g(x)$, if it exists.

Step 1 Find the left-hand limit. For $x < 2$, we use $g(x) = x + 1$:

$$\lim_{x \to 2^-} g(x) = \lim_{x \to 2^-} (x + 1) = 2 + 1 = 3$$

Step 2 Find the right-hand limit. For $x > 2$, we use $g(x) = 7 - x$:

$$\lim_{x \to 2^+} g(x) = \lim_{x \to 2^+} (7 - x) = 7 - 2 = 5$$

Step 3 Compare. Since $3 \neq 5$, the one-sided limits are not equal. Therefore, $\displaystyle\lim_{x \to 2} g(x)$ does not exist.

Note that $g(2) = 5$, but this is irrelevant to whether the limit exists. The limit fails to exist because the function approaches different values from the left and right.

1.2 Evaluating Limits

Now that we understand what limits mean, we need reliable techniques for computing them. The AP Calculus exam tests several core methods, which we cover here in increasing order of sophistication.

Direct Substitution

The simplest and most powerful technique: if the function is continuous at $x = a$ (we will formalize this in Section 1.5), then

$$\lim_{x \to a} f(x) = f(a)$$

This works for all polynomials, rational functions (where the denominator is nonzero), trigonometric functions, exponentials, and logarithms at points in their domains. Always try direct substitution first.

Worked Example 1.2a — Direct Substitution

Evaluate $\displaystyle\lim_{x \to 3} (2x^2 - 5x + 1)$.

Solution Since polynomials are continuous everywhere, we substitute directly:

$$\lim_{x \to 3} (2x^2 - 5x + 1) = 2(3)^2 - 5(3) + 1 = 18 - 15 + 1 = 4$$

Factoring and Canceling

When direct substitution yields the indeterminate form $\frac{0}{0}$, the numerator and denominator share a common factor of $(x - a)$. Factor it out, cancel, and then substitute.

Worked Example 1.2b — Factoring

Evaluate $\displaystyle\lim_{x \to 4} \frac{x^2 - 16}{x - 4}$.

Step 1 Try direct substitution: $\dfrac{4^2 - 16}{4 - 4} = \dfrac{0}{0}$. This is indeterminate, so we need another approach.

Step 2 Factor the numerator using the difference of squares:

$$\frac{x^2 - 16}{x - 4} = \frac{(x-4)(x+4)}{x-4}$$

Step 3 Cancel the common factor. For $x \neq 4$:

$$\frac{(x-4)(x+4)}{x-4} = x + 4$$

Step 4 Now substitute:

$$\lim_{x \to 4} (x + 4) = 4 + 4 = 8$$

Rationalization (Conjugate Method)

When you encounter $\frac{0}{0}$ and the expression involves square roots, multiply the numerator and denominator by the conjugate of the expression containing the radical. Recall that the conjugate of $\sqrt{a} - b$ is $\sqrt{a} + b$.

Worked Example 1.2c — Rationalization

Evaluate $\displaystyle\lim_{x \to 0} \frac{\sqrt{x+9} - 3}{x}$.

Step 1 Direct substitution gives $\dfrac{\sqrt{9} - 3}{0} = \dfrac{0}{0}$. Indeterminate.

Step 2 Multiply by the conjugate of the numerator:

$$\frac{\sqrt{x+9} - 3}{x} \cdot \frac{\sqrt{x+9} + 3}{\sqrt{x+9} + 3} = \frac{(x+9) - 9}{x(\sqrt{x+9} + 3)} = \frac{x}{x(\sqrt{x+9} + 3)}$$

Step 3 Cancel $x$ (valid for $x \neq 0$):

$$= \frac{1}{\sqrt{x+9} + 3}$$

Step 4 Now substitute $x = 0$:

$$\lim_{x \to 0} \frac{1}{\sqrt{x+9} + 3} = \frac{1}{\sqrt{9} + 3} = \frac{1}{3 + 3} = \frac{1}{6}$$

1.3 Limits Involving Infinity

So far we have considered limits where $x$ approaches a finite number. We now extend the concept in two important ways: what happens as $x$ grows without bound, and what happens when function values grow without bound.

Limits at Infinity and Horizontal Asymptotes

Definition: Limit at Infinity

We write $\displaystyle\lim_{x \to \infty} f(x) = L$ if the values of $f(x)$ can be made arbitrarily close to $L$ by taking $x$ sufficiently large. If this limit exists and equals $L$, then $y = L$ is a horizontal asymptote of the graph of $f$.

For rational functions $\dfrac{p(x)}{q(x)}$ where $p$ and $q$ are polynomials, the behavior as $x \to \infty$ depends on the degrees of the numerator and denominator:

  • degree of $p$ < degree of $q$: the limit is $0$.
  • degree of $p$ = degree of $q$: the limit is the ratio of leading coefficients.
  • degree of $p$ > degree of $q$: the limit is $\pm\infty$ (no horizontal asymptote).

Worked Example 1.3a — Horizontal Asymptote

Evaluate $\displaystyle\lim_{x \to \infty} \frac{5x^2 - 3x + 1}{2x^2 + 7}$.

Step 1 The numerator and denominator are both degree 2. Divide every term by $x^2$:

$$\frac{5x^2 - 3x + 1}{2x^2 + 7} = \frac{5 - \frac{3}{x} + \frac{1}{x^2}}{2 + \frac{7}{x^2}}$$

Step 2 As $x \to \infty$, all terms with $x$ in the denominator approach $0$:

$$\lim_{x \to \infty} \frac{5 - \frac{3}{x} + \frac{1}{x^2}}{2 + \frac{7}{x^2}} = \frac{5 - 0 + 0}{2 + 0} = \frac{5}{2}$$

Therefore $y = \frac{5}{2}$ is a horizontal asymptote.

Infinite Limits and Vertical Asymptotes

Definition: Vertical Asymptote

The line $x = a$ is a vertical asymptote of $f$ if at least one of the following is true:

$$\lim_{x \to a^+} f(x) = \pm\infty \quad \text{or} \quad \lim_{x \to a^-} f(x) = \pm\infty$$

When we write $\displaystyle\lim_{x \to a} f(x) = \infty$, we mean the function values increase without bound. Strictly speaking, the limit does not exist (since $\infty$ is not a real number), but this notation tells us how the limit fails to exist.

Worked Example 1.3b — Vertical Asymptote

Find all vertical asymptotes of $f(x) = \dfrac{x + 1}{x^2 - 4}$ and evaluate the relevant one-sided limits.

Step 1 Factor the denominator: $x^2 - 4 = (x-2)(x+2)$. The denominator is zero at $x = 2$ and $x = -2$.

Step 2 Check whether the numerator is also zero at these points. At $x = 2$: the numerator is $2 + 1 = 3 \neq 0$. At $x = -2$: the numerator is $-2 + 1 = -1 \neq 0$. Since neither cancels, both are vertical asymptotes.

Step 3 Analyze the sign near $x = 2$. Write $f(x) = \dfrac{x+1}{(x-2)(x+2)}$.

  • As $x \to 2^+$: numerator $\to 3 > 0$, $(x-2) \to 0^+$, $(x+2) \to 4 > 0$. So $f(x) \to +\infty$.
  • As $x \to 2^-$: numerator $\to 3 > 0$, $(x-2) \to 0^-$, $(x+2) \to 4 > 0$. So $f(x) \to -\infty$.

Therefore $\displaystyle\lim_{x \to 2^+} f(x) = +\infty$ and $\displaystyle\lim_{x \to 2^-} f(x) = -\infty$. By similar analysis, $x = -2$ is also a vertical asymptote.

1.4 The Squeeze Theorem

Some limits cannot be evaluated by direct substitution or algebraic manipulation. The Squeeze Theorem (also called the Sandwich Theorem or Pinching Theorem) provides a way to evaluate limits by trapping the function between two simpler functions whose limits we already know.

Theorem: The Squeeze Theorem

Suppose $g(x) \leq f(x) \leq h(x)$ for all $x$ in some open interval containing $a$ (except possibly at $a$ itself). If

$$\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L$$

then $\displaystyle\lim_{x \to a} f(x) = L$ as well.

The intuition is simple: if $f$ is squeezed between $g$ and $h$, and both $g$ and $h$ converge to $L$, then $f$ has no choice but to converge to $L$ as well.

The Squeeze Theorem in action: x·sin(1/x) is trapped between -|x| and |x|, so it must approach 0 as x → 0.

The Most Important Limit in Calculus

The Squeeze Theorem gives us one of the most fundamental results needed for calculus:

Theorem

$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$

Proof sketch. For $0 < x < \frac{\pi}{2}$, one can show using a unit circle argument that

$$\cos x \leq \frac{\sin x}{x} \leq 1$$

Since $\displaystyle\lim_{x \to 0} \cos x = 1$ and $\displaystyle\lim_{x \to 0} 1 = 1$, the Squeeze Theorem gives $\displaystyle\lim_{x \to 0} \frac{\sin x}{x} = 1$.

A companion result that appears frequently on the AP exam is:

$$\lim_{x \to 0} \frac{1 - \cos x}{x} = 0$$

Worked Example 1.4 — Squeeze Theorem

Evaluate $\displaystyle\lim_{x \to 0}\, x^2 \cos\!\left(\frac{1}{x}\right)$.

Step 1 Direct substitution fails because $\cos(1/x)$ oscillates wildly as $x \to 0$. However, we know that for all $x \neq 0$:

$$-1 \leq \cos\!\left(\frac{1}{x}\right) \leq 1$$

Step 2 Multiply the entire inequality by $x^2$ (which is non-negative, so the inequalities are preserved):

$$-x^2 \leq x^2 \cos\!\left(\frac{1}{x}\right) \leq x^2$$

Step 3 Evaluate the limits of the bounding functions:

$$\lim_{x \to 0} (-x^2) = 0 \quad \text{and} \quad \lim_{x \to 0} x^2 = 0$$

Step 4 By the Squeeze Theorem:

$$\lim_{x \to 0}\, x^2 \cos\!\left(\frac{1}{x}\right) = 0$$

1.5 Continuity

We now use limits to make precise the intuitive idea that a function is "continuous" if you can draw its graph without lifting your pen. While this intuition is helpful, the formal definition is what allows us to prove powerful theorems.

Definition: Continuity at a Point

A function $f$ is continuous at $x = a$ if all three of the following conditions hold:

  1. $f(a)$ is defined (i.e., $a$ is in the domain of $f$).
  2. $\displaystyle\lim_{x \to a} f(x)$ exists.
  3. $\displaystyle\lim_{x \to a} f(x) = f(a)$.

If any one of these conditions fails, $f$ is discontinuous at $x = a$.

Types of Discontinuities

Discontinuities are classified based on how the conditions above fail:

  • Removable discontinuity: The limit $\displaystyle\lim_{x \to a} f(x) = L$ exists, but either $f(a)$ is undefined or $f(a) \neq L$. The "hole" can be filled by defining or redefining $f(a) = L$. Example: $f(x) = \dfrac{x^2 - 1}{x - 1}$ at $x = 1$.
  • Jump discontinuity: Both one-sided limits exist, but they are not equal: $\displaystyle\lim_{x \to a^-} f(x) \neq \lim_{x \to a^+} f(x)$. The function "jumps" from one value to another. This commonly occurs in piecewise functions.
  • Infinite discontinuity: At least one of the one-sided limits is $\pm\infty$, indicating a vertical asymptote. Example: $f(x) = \dfrac{1}{x}$ at $x = 0$.

Explore different types of discontinuities: adjust the slider to see how changing f(a) affects continuity at x = a.

Worked Example 1.5a — Testing Continuity

Determine whether $f(x) = \begin{cases} \dfrac{x^2 - 9}{x - 3}, & x \neq 3 \\[6pt] 7, & x = 3 \end{cases}$ is continuous at $x = 3$.

Step 1 Is $f(3)$ defined? Yes, $f(3) = 7$. Condition 1 is satisfied.

Step 2 Does the limit exist? For $x \neq 3$:

$$\frac{x^2 - 9}{x - 3} = \frac{(x-3)(x+3)}{x-3} = x + 3$$

$$\lim_{x \to 3} f(x) = \lim_{x \to 3} (x + 3) = 6$$

The limit exists and equals $6$. Condition 2 is satisfied.

Step 3 Does the limit equal the function value? We need $\displaystyle\lim_{x \to 3} f(x) = f(3)$, i.e., $6 = 7$. This is false. Condition 3 fails.

Therefore $f$ is not continuous at $x = 3$. This is a removable discontinuity because we could fix it by redefining $f(3) = 6$.

The Intermediate Value Theorem

Continuous functions have a remarkable property: they cannot skip over values. This is formalized by one of the most important theorems in calculus.

Theorem: Intermediate Value Theorem (IVT)

If $f$ is continuous on the closed interval $[a, b]$ and $N$ is any number between $f(a)$ and $f(b)$, then there exists at least one value $c$ in $(a, b)$ such that $f(c) = N$.

The IVT is an existence theorem: it guarantees that a solution exists, but it does not tell you how to find it. On the AP exam, the IVT is commonly used to prove that a function has a root (a zero) on an interval.

Worked Example 1.5b — Applying the IVT

Show that the equation $x^3 + x - 1 = 0$ has a solution in the interval $(0, 1)$.

Step 1 Define $f(x) = x^3 + x - 1$. This is a polynomial, so it is continuous everywhere, and in particular on $[0, 1]$.

Step 2 Evaluate $f$ at the endpoints:

  • $f(0) = 0 + 0 - 1 = -1$
  • $f(1) = 1 + 1 - 1 = 1$

Step 3 Since $f(0) = -1 < 0 < 1 = f(1)$, the value $N = 0$ lies between $f(0)$ and $f(1)$.

Step 4 By the IVT, there exists at least one value $c \in (0, 1)$ such that $f(c) = 0$. Therefore $x^3 + x - 1 = 0$ has a solution in $(0, 1)$.

1.6 Practice Problems

Test your understanding with the following problems. Each one has a detailed solution you can reveal after attempting the problem on your own.

Problem 1

Evaluate $\displaystyle\lim_{x \to 5} \frac{x^2 - 25}{x - 5}$.

Show Solution

Direct substitution gives $\frac{0}{0}$. Factor the numerator:

$$\frac{x^2 - 25}{x - 5} = \frac{(x-5)(x+5)}{x-5} = x + 5 \quad (x \neq 5)$$

$$\lim_{x \to 5} (x + 5) = 10$$

Problem 2

Evaluate $\displaystyle\lim_{x \to 0} \frac{\sqrt{1+x} - 1}{x}$.

Show Solution

Direct substitution gives $\frac{0}{0}$. Multiply by the conjugate:

$$\frac{\sqrt{1+x} - 1}{x} \cdot \frac{\sqrt{1+x} + 1}{\sqrt{1+x} + 1} = \frac{(1+x) - 1}{x(\sqrt{1+x} + 1)} = \frac{x}{x(\sqrt{1+x} + 1)} = \frac{1}{\sqrt{1+x} + 1}$$

$$\lim_{x \to 0} \frac{1}{\sqrt{1+x} + 1} = \frac{1}{\sqrt{1} + 1} = \frac{1}{2}$$

Problem 3

Evaluate $\displaystyle\lim_{x \to \infty} \frac{3x^3 - 2x}{7x^3 + 4x^2 - 1}$.

Show Solution

The numerator and denominator are both degree 3. Divide every term by $x^3$:

$$\frac{3 - \frac{2}{x^2}}{7 + \frac{4}{x} - \frac{1}{x^3}}$$

As $x \to \infty$, all fractional terms vanish:

$$\lim_{x \to \infty} \frac{3 - 0}{7 + 0 - 0} = \frac{3}{7}$$

Problem 4

Evaluate $\displaystyle\lim_{x \to 0} \frac{\sin(3x)}{x}$.

Show Solution

We use the known result $\displaystyle\lim_{u \to 0} \frac{\sin u}{u} = 1$ by rewriting the expression:

$$\frac{\sin(3x)}{x} = 3 \cdot \frac{\sin(3x)}{3x}$$

Let $u = 3x$. As $x \to 0$, we have $u \to 0$:

$$\lim_{x \to 0} 3 \cdot \frac{\sin(3x)}{3x} = 3 \cdot 1 = 3$$

Problem 5

Let $f(x) = \begin{cases} 2x + 1, & x < 1 \\ x^2 + c, & x \geq 1 \end{cases}$. Find the value of $c$ that makes $f$ continuous at $x = 1$.

Show Solution

For continuity at $x = 1$, we need $\displaystyle\lim_{x \to 1^-} f(x) = \lim_{x \to 1^+} f(x) = f(1)$.

Left-hand limit: $\displaystyle\lim_{x \to 1^-} (2x + 1) = 2(1) + 1 = 3$

Right-hand limit (and function value): $\displaystyle\lim_{x \to 1^+} (x^2 + c) = 1 + c$, and $f(1) = 1 + c$.

Set them equal: $1 + c = 3$, so $c = 2$.

Problem 6

Evaluate $\displaystyle\lim_{x \to 2} \frac{x^3 - 8}{x^2 - 4}$.

Show Solution

Direct substitution gives $\frac{0}{0}$. Factor both:

$$x^3 - 8 = (x - 2)(x^2 + 2x + 4), \quad x^2 - 4 = (x-2)(x+2)$$

$$\frac{(x-2)(x^2 + 2x + 4)}{(x-2)(x+2)} = \frac{x^2 + 2x + 4}{x + 2} \quad (x \neq 2)$$

$$\lim_{x \to 2} \frac{x^2 + 2x + 4}{x + 2} = \frac{4 + 4 + 4}{4} = \frac{12}{4} = 3$$

Problem 7

Use the Squeeze Theorem to evaluate $\displaystyle\lim_{x \to \infty} \frac{\sin x}{x}$.

Show Solution

We know that $-1 \leq \sin x \leq 1$ for all $x$. Dividing by $x$ (positive for large $x$):

$$-\frac{1}{x} \leq \frac{\sin x}{x} \leq \frac{1}{x}$$

Since $\displaystyle\lim_{x \to \infty} \left(-\frac{1}{x}\right) = 0$ and $\displaystyle\lim_{x \to \infty} \frac{1}{x} = 0$, the Squeeze Theorem gives:

$$\lim_{x \to \infty} \frac{\sin x}{x} = 0$$

Problem 8

Show that the equation $e^x = 3 - x$ has at least one solution on the interval $(0, 1)$.

Show Solution

Define $f(x) = e^x - (3 - x) = e^x + x - 3$. This function is continuous on $[0, 1]$ since it is the sum of continuous functions.

Evaluate at the endpoints:

  • $f(0) = e^0 + 0 - 3 = 1 - 3 = -2 < 0$
  • $f(1) = e^1 + 1 - 3 = e - 2 \approx 0.718 > 0$

Since $f(0) < 0 < f(1)$, by the Intermediate Value Theorem, there exists $c \in (0, 1)$ such that $f(c) = 0$, i.e., $e^c = 3 - c$.

Problem 9

Evaluate $\displaystyle\lim_{h \to 0} \frac{(2+h)^3 - 8}{h}$.

Show Solution

Expand $(2+h)^3$:

$$(2+h)^3 = 8 + 12h + 6h^2 + h^3$$

Substitute:

$$\frac{(8 + 12h + 6h^2 + h^3) - 8}{h} = \frac{12h + 6h^2 + h^3}{h} = 12 + 6h + h^2$$

$$\lim_{h \to 0} (12 + 6h + h^2) = 12$$

Note: this is actually the derivative of $x^3$ evaluated at $x = 2$, which you will study in Chapter 2.

Problem 10

Find all vertical and horizontal asymptotes of $f(x) = \dfrac{2x^2 + 3}{x^2 - 1}$.

Show Solution

Vertical asymptotes: Set the denominator equal to zero: $x^2 - 1 = 0 \Rightarrow x = \pm 1$. The numerator at these points: $2(1) + 3 = 5 \neq 0$ and $2(1) + 3 = 5 \neq 0$. Since the numerator is nonzero, both $x = 1$ and $x = -1$ are vertical asymptotes.

Horizontal asymptote: Numerator and denominator are both degree 2. The ratio of leading coefficients is $\frac{2}{1} = 2$.

$$\lim_{x \to \pm\infty} \frac{2x^2 + 3}{x^2 - 1} = 2$$

So $y = 2$ is the horizontal asymptote.
