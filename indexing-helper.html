Google Search Console 색인 요청을 빠르게 처리하세요

📋 사용 방법

  1. 아래 목록에서 색인을 생성할 페이지를 찾으세요
  2. "📋 URL 복사" 버튼을 클릭하세요 (자동으로 클립보드에 복사됨)
  3. "🔍 Search Console 열기" 버튼을 클릭하세요
  4. 열린 페이지 상단 검색창에 붙여넣기 (Ctrl+V 또는 Cmd+V)
  5. Enter를 누른 후 "색인 생성 요청" 버튼을 클릭하세요
  6. 모든 페이지에 대해 반복하세요
