# marketing/ — 사이트 홍보 자료

> MathHub(mathhub.io) 소개·홍보용 원고. **이 폴더는 사이트에 배포되지 않습니다**
> (`scripts/deploy.sh` 의 EXCLUDE 에 포함). 블로그·SNS에 가져다 쓰는 소스 모음.

## 파일

| 파일 | 용도 |
|------|------|
| `landing-ko.html` | 한국어 **독립 랜딩 페이지**. 브라우저로 바로 열거나 링크 공유용. 자체 완결(인라인 CSS). |
| `landing-en.html` | 영어 독립 랜딩 페이지. 글로벌 SNS 공유용. |
| `blog-article-ko.html` | 한국어 **블로그 본문 붙여넣기용**. 티스토리/네이버/워드프레스 HTML 편집기에 그대로 붙여넣기(인라인 스타일이라 CSS 제거돼도 유지). |
| `blog-article-en.html` | 영어 블로그 본문 붙여넣기용(WordPress/Medium/Blogger 등). |
| `sns-posts.md` | 인스타·X·스레드·카카오스토리·페북 등 **플랫폼별 짧은 카피**(한/영). |

## 쓰는 법
1. **블로그 글**: `blog-article-*.html` 내용을 블로그 글쓰기 → HTML 모드에 붙여넣기.
2. **랜딩 페이지**: `landing-*.html` 을 그대로 호스팅하거나 링크로 공유.
3. **SNS**: `sns-posts.md` 의 해당 플랫폼 카피 복사 + 풀이 페이지 스크린샷 첨부.

## 카피 원칙
- 사이트의 **실제** 제공 내용만 사용(직접 쓴 교재, 단계별 풀이, 공식 기출, 무료·가입불필요, 다국가 커리큘럼). 과장·허위 금지.
- 링크에 `utm_source`(채널) 를 넣어 유입을 GSC/Analytics에서 구분.
- 핵심 메시지: **"정답이 아니라 풀이를 설명한다 + 전부 무료"**.
