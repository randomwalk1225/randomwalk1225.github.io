1. Rappel — Fonction du 1er degré

📖 Définition

Une fonction du 1er degré (ou fonction affine) est de la forme :

$$f(x) = ax + b$$

où $a$ est le taux de variation (pente) et $b$ est l'ordonnée à l'origine.

📈 Si $a > 0$

La droite est croissante : monte de gauche à droite.

📉 Si $a < 0$

La droite est décroissante : descend de gauche à droite.

✏️ Exemple 1

Tracer le graphique de $f(x) = 2x - 3$ et identifier ses caractéristiques.

Solution :
  • Pente : $a = 2$ (croissante, monte de 2 pour chaque déplacement de 1 vers la droite)
  • Ordonnée à l'origine : $b = -3$ → point $(0,\,-3)$
  • Zéro : $2x - 3 = 0 \Rightarrow x = \dfrac{3}{2} = 1{,}5$ → point $(1{,}5,\;0)$

Graphique de $f(x) = 2x - 3$ : droite croissante, zéro en $x = 1{,}5$

2. Fonction du 2e degré (quadratique)

📖 Définition

Une fonction quadratique (du 2e degré) est de la forme :

$$f(x) = ax^2 + bx + c \quad (a \neq 0)$$

Son graphique est une parabole.

🔼 Si $a > 0$ — parabole ouverte vers le haut

Le sommet est un minimum. La courbe descend puis remonte.

🔽 Si $a < 0$ — parabole ouverte vers le bas

Le sommet est un maximum. La courbe monte puis redescend.

⭐ Propriétés fondamentales de la parabole

  • La parabole est symétrique par rapport à son axe de symétrie.
  • Le point le plus bas (ou le plus haut) s'appelle le sommet $S(h, k)$.
  • L'axe de symétrie est la droite verticale $x = h$.
  • Plus $|a|$ est grand, plus la parabole est étroite.

3. Forme canonique

📖 Forme canonique

$$f(x) = a(x - h)^2 + k$$
  • $S(h, k)$ est le sommet de la parabole.
  • L'axe de symétrie est $x = h$.
  • $a > 0$ : minimum en $k$ ; $a < 0$ : maximum en $k$.

✏️ Exemple 2

Soit $f(x) = 2(x - 3)^2 - 8$. Identifier le sommet, l'axe de symétrie, et les coordonnées de deux autres points.

Solution :
  • Forme canonique avec $a = 2$, $h = 3$, $k = -8$.
  • Sommet : $S(3,\,-8)$.
  • Axe de symétrie : $x = 3$.
  • En $x = 4$ : $f(4) = 2(1)^2 - 8 = -6$ → point $(4,\,-6)$.
  • Par symétrie, en $x = 2$ : $f(2) = -6$ → point $(2,\,-6)$.
  • Comme $a = 2 > 0$, le sommet est un minimum.

Parabole $f(x) = 2(x-3)^2 - 8$ : sommet en $(3, -8)$, axe de symétrie $x = 3$

4. Forme générale et conversion

📖 Forme générale

$$f(x) = ax^2 + bx + c$$

Le sommet peut se trouver par la formule :

$$h = -\frac{b}{2a}, \qquad k = f(h)$$

✏️ Exemple 3 — Trouver le sommet à partir de la forme générale

Soit $f(x) = x^2 - 6x + 5$. Trouver le sommet et l'axe de symétrie.

Solution :
1Identifier $a = 1$, $b = -6$, $c = 5$.
2Abscisse du sommet : $h = -\dfrac{b}{2a} = -\dfrac{-6}{2} = 3$.
3Ordonnée du sommet : $k = f(3) = 9 - 18 + 5 = -4$.
4Sommet : $S(3, -4)$ ; axe de symétrie : $x = 3$.
5Forme canonique équivalente : $f(x) = (x-3)^2 - 4$.

💡 Méthode de complétion du carré

On peut aussi convertir en forme canonique par complétion du carré :

$$x^2 - 6x + 5 = (x^2 - 6x + 9) - 9 + 5 = (x-3)^2 - 4$$

Ajouter et soustraire $\left(\dfrac{b}{2a}\right)^2 = 9$ pour créer un carré parfait.

5. Zéros d'une fonction quadratique

Les zéros (ou racines) d'une fonction quadratique sont les valeurs de $x$ pour lesquelles $f(x) = 0$. Graphiquement, ce sont les abscisses des points d'intersection avec l'axe des $x$.

⭐ Formule quadratique (formule générale)

Pour $ax^2 + bx + c = 0$ :

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Le discriminant $\Delta = b^2 - 4ac$ détermine le nombre de solutions :

  • $\Delta > 0$ : deux zéros distincts (la parabole coupe l'axe des $x$ en deux points)
  • $\Delta = 0$ : un seul zéro (tangente à l'axe des $x$)
  • $\Delta < 0$ : aucun zéro réel (pas d'intersection avec l'axe des $x$)

✏️ Exemple 4 — Trouver les zéros

Trouver les zéros de $f(x) = x^2 - 6x + 5$.

Solution :
1Calculer $\Delta = (-6)^2 - 4(1)(5) = 36 - 20 = 16 > 0$ → deux zéros.
2$x = \dfrac{6 \pm \sqrt{16}}{2} = \dfrac{6 \pm 4}{2}$
3$x_1 = \dfrac{6+4}{2} = 5$ et $x_2 = \dfrac{6-4}{2} = 1$.

Vérification par factorisation : $f(x) = (x-1)(x-5)$ ✓

⚠️ Erreur fréquente

Ne pas confondre les zéros (valeurs de $x$) avec l'ordonnée à l'origine ($y$-intercept, valeur en $x=0$). Les zéros sont sur l'axe des $x$, l'ordonnée à l'origine est sur l'axe des $y$.

Récapitulatif

Forme Expression Avantage
Générale $ax^2 + bx + c$ Ordonnée à l'origine directe ($c$)
Canonique $a(x-h)^2 + k$ Sommet $S(h,k)$ visible
Factorisée $a(x-x_1)(x-x_2)$ Zéros visibles ($x_1$ et $x_2$)

6. Exercices

📝 Exercices — Fonctions du 1er et 2e degré

  1. Soit $f(x) = 3x - 6$.
    1. Trouver le zéro de $f$.
    2. Quel est l'ordonnée à l'origine ?
    3. La fonction est-elle croissante ou décroissante ?
  2. Pour la parabole $f(x) = -(x+2)^2 + 9$ :
    1. Identifier le sommet et l'axe de symétrie.
    2. La parabole est-elle ouverte vers le haut ou vers le bas ?
    3. Trouver les zéros (sans la formule quadratique, par algebre).
  3. Soit $g(x) = 2x^2 + 8x - 10$.
    1. Trouver les coordonnées du sommet.
    2. Calculer le discriminant et déterminer le nombre de zéros.
    3. Trouver les zéros à l'aide de la formule quadratique.
  4. Un artisan lance une balle. La hauteur (en mètres) en fonction du temps $t$ (en secondes) est donnée par $h(t) = -5t^2 + 20t + 1$.
    1. Quelle est la hauteur maximale atteinte par la balle ?
    2. À quel instant la balle atteint-elle cette hauteur maximale ?
    3. Quand la balle revient-elle au niveau du sol ($h = 0$) ?
🔑 Voir les réponses
    1. $3x - 6 = 0 \Rightarrow x = 2$
    2. $f(0) = -6$
    3. Croissante ($a = 3 > 0$)
    1. Sommet $S(-2, 9)$ ; axe $x = -2$
    2. Ouverte vers le bas ($a = -1 < 0$)
    3. $-(x+2)^2 + 9 = 0 \Rightarrow (x+2)^2 = 9 \Rightarrow x+2 = \pm 3 \Rightarrow x = 1$ ou $x = -5$
    1. $h = -\frac{8}{4} = -2$, $k = 2(4) - 16 - 10 = -18$ → Sommet $S(-2, -18)$
    2. $\Delta = 64 + 80 = 144 > 0$ → deux zéros
    3. $x = \frac{-8 \pm 12}{4}$ → $x = 1$ ou $x = -5$
    1. $t = -\frac{20}{-10} = 2$ s → $h(2) = -20 + 40 + 1 = 21$ m
    2. $t = 2$ s
    3. $-5t^2 + 20t + 1 = 0$ → $\Delta = 400 + 20 = 420$ → $t = \frac{-20 \pm \sqrt{420}}{-10} \approx 4{,}05$ s
